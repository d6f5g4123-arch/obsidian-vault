# 常见概率分布：均匀分布与泊松分布详解

**核心概念定义**

- **概率分布**是随机变量所有可能取值的概率分配规律。
- **均匀分布**：在某个给定区间内，各取值出现的概率完全相等的分布。
- **泊松分布**：描述单位时间或单位空间内某事件发生次数的离散概率分布。

**详细说明**

### 1. 均匀分布

均匀分布有两种类型：
- **离散均匀分布**：有限个取值，每个值概率相等；常见于掷骰子等例子。
- **连续均匀分布**：定义在区间 $[a, b]$ 上，区间内任一点出现的概率一致。

**概率密度函数（连续）**：
$$
f(x) = \begin{cases}
\frac{1}{b-a}, & a \leq x \leq b \\
0, & \text{otherwise}
\end{cases}
$$

**期望与方差**：
- 期望：$\mathrm{E}[X] = \frac{a+b}{2}$
- 方差：$\mathrm{Var}(X) = \frac{(b-a)^2}{12}$

### 2. 泊松分布

泊松分布常用于描述在单位时间或空间内，稀疏独立事件的发生次数（如每小时到达银行的顾客数）。

**概率质量函数**：
$$
P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!}, \quad k = 0, 1, 2, ...
$$
其中 $\lambda$ 表示单位时间（或空间）内事件的平均发生次数。

**期望与方差**：
- 期望：$\mathrm{E}[X] = \lambda$
- 方差：$\mathrm{Var}(X) = \lambda$

**关键公式/代码**

- 均匀分布随机采样（Python 示例）：
```python
import numpy as np
# 连续均匀分布采样
samples = np.random.uniform(a, b, size=1000)
# 离散均匀分布采样
samples_2 = np.random.randint(low, high + 1, size=1000)
```
- 泊松分布采样（Python 示例）：
```python
import numpy as np
samples = np.random.poisson(lam=λ, size=1000)
```

**例题/应用**

1. **均匀分布例题**：
    - 设随机变量 $X$ 服从区间 $[2, 5]$ 上的均匀分布，则 $P(3 < X < 4) = \frac{4-3}{5-2} = \frac{1}{3}$。

2. **泊松分布例题**：
    - 某超市每分钟平均有 2 位顾客到来，求恰好 3 位顾客到来的概率。
    $$
    P(X=3) = \frac{2^3 \cdot e^{-2}}{3!} \approx 0.180
    $$

**易混淆点**

- **均匀分布与正态分布区别**：均匀分布概率分布是平的，正态分布则呈钟形。
- **泊松分布与二项分布的关系**：当二项分布 $n$ 很大，$p$ 很小且 $n p = \lambda$ 恒定时，二项分布近似为泊松分布。
- **均匀分布分为连续与离散，两者公式不同**。
