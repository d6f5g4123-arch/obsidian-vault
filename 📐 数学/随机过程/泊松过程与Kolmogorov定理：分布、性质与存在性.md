# 泊松过程与Kolmogorov定理：分布、性质与存在性

## 核心概念定义

1.  **泊松过程 (Poisson Process)**
    - 一种重要的**计数过程**，用于描述在单位时间或单位空间内，某事件发生次数的随机过程。其核心特征是事件的发生是独立且平稳的。

2.  **Kolmogorov 定理 (Kolmogorov's Theorem)**
    - 由苏联数学家安德雷·柯尔莫哥洛夫提出，是**随机过程存在性**的奠基性定理。它指出，如果一个分布函数族满足**对称性**和**相容性**，则必存在唯一的随机过程，其所有有限维分布恰好就是这个函数族。

3.  **均匀分布 (Uniform Distribution)**
    - 在某个区间内所有取值出现的概率相等的概率分布。在泊松过程的背景下，**单粒子释放时刻**在给定条件下服从均匀分布。

## 详细说明与关键性质

### 一、泊松分布及其极限形式

泊松过程在固定时间 $t$ 内的计数 $N(t)$ 服从**泊松分布**。

- **概率质量函数 (PMF)**：
  $$
  P(N(t)=k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}, \quad k = 0, 1, 2, \ldots
  $$
  其中 $\lambda$ 是事件发生的平均速率。

- **泊松定理 (极限形式)**：当二项分布的试验次数 $n$ 很大而成功概率 $p$ 很小时，泊松分布可作为其近似：
  $$
  \lim_{n\to\infty} C_n^k p^k (1-p)^{n-k} = \frac{(\lambda)^k}{k!} e^{-\lambda}, \quad \text{其中 } np \to \lambda.
  $$

### 二、泊松过程的性质

1.  **独立增量性**：在不相交的时间区间内，事件发生的次数是相互独立的。
2.  **平稳增量性**：在任意长度相同的时间区间内，事件发生次数的概率分布只依赖于区间的长度，而与起点无关。
3.  **协方差性质**：对于泊松过程 $N(t)$，其协方差为：
    $$
    \mathrm{cov}(N(s), N(t)) = \lambda \min(s, t).
    $$
    推导利用了独立增量性：$\mathrm{cov}(N(s), N(t)) = \mathrm{cov}(N(s), N(s) + (N(t)-N(s))) = \mathrm{Var}(N(s)) = \lambda s$（当 $s \le t$ 时）。
4.  **可加性**：两个独立的泊松过程 $N_1(t) \sim P(\lambda_1 t)$ 和 $N_2(t) \sim P(\lambda_2 t)$ 之和仍是一个泊松过程：
    $$
    N_1(t) + N_2(t) \sim P((\lambda_1+\lambda_2)t).
    $$
    ==这是泊松分布的一个重要特性，在合并独立事件流时非常有用。==

### 三、Kolmogorov 一致性定理（存在性定理）

该定理是构建随机过程的数学基础。

- **定理陈述**：设给定了一个分布函数族 $\{F_{t_1, \ldots, t_n}(x_1, \ldots, x_n): t_i \in T, n \ge 1\}$。如果这个函数族满足：
  1.  **对称性**：对 $(t_1, \ldots, t_n)$ 的任意排列，相应的有限维分布是对称的。
  2.  **相容性**：若 $m < n$，则 $F_{t_1, \ldots, t_m}(x_1, \ldots, x_m)$ 是 $F_{t_1, \ldots, t_n}(x_1, \ldots, x_n)$ 的边际分布。
- **结论**：则存在一个概率空间 $ (\Omega, \mathcal{F}, P) $ 及其上的随机过程 $\{X(t), t \in T\}$，使得该过程的所有有限维分布恰好就是给定的分布函数族：
  $$
  F_{t_1, \ldots, t_n}(x_1, \ldots, x_n) = P(X(t_1) \le x_1, \ldots, X(t_n) \le x_n).
  $$
  ==此定理表明，只要有限维分布相容，整个随机过程就唯一确定了。这是从局部（有限维）定义全局（过程）的关键。==

### 四、泊松过程中的事件时刻分布

考虑一个泊松过程，已知在 $[0, t]$ 内仅发生了一次事件（即 $N(t)=1$），记该事件的到达时刻为 $S_1$。

- **条件分布**：在 $N(t)=1$ 的条件下，$S_1$ 在 $[0, t]$ 上服从**均匀分布**。
  $$
  P(S_1 \le s \mid N(t)=1) = \frac{s}{t}, \quad 0 \le s \le t.
  $$
- **推导**：利用泊松过程的独立增量性，
  $$
  P(S_1 \le s \mid N(t)=1) = \frac{P(N(s)=1, N(t)-N(s)=0)}{P(N(t)=1)} = \frac{(\lambda s e^{-\lambda s}) \cdot e^{-\lambda (t-s)}}{\lambda t e^{-\lambda t}} = \frac{s}{t}.
  $$
  ==这一结论直观上意味着，在只知道发生了一次事件的条件下，这次事件在时间区间内是“等可能”出现在任何位置的。==

### 五、联合分布函数及其性质

对于随机向量 $(X, Y)$，其**联合分布函数**定义为 $F(x, y) = P(X \le x, Y \le y)$。

- **与互补事件概率的关系**：定义 $G(x, y) = P(X > x, Y \le y)$，则有 $G(x, y) = P(Y \le y) - F(x, y)$。
- **密度函数关系**：若联合密度 $f(x, y)$ 存在，则 $f(x, y) = \frac{\partial^2 F}{\partial x \partial y}$。同时，对 $G$ 求导可得 $\frac{\partial^2 G}{\partial y \partial x} = -\frac{\partial^2 F}{\partial y \partial x}$。
- **高维推广**：对于 $n$ 维随机向量，类似的关系可以通过对分布函数 $F$ 和由“大于”和“小于”条件定义的函数 $G_k$ 求高阶混合偏导得到，符号由条件中“大于”号的个数决定。

## 关键公式总结

1.  **泊松分布**：$P(N(t)=k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}$
2.  **泊松过程协方差**：$\mathrm{cov}(N(s), N(t)) = \lambda \min(s, t)$
3.  **独立泊松过程之和**：$N_1(t)+N_2(t) \sim P((\lambda_1+\lambda_2)t)$
4.  **条件均匀分布**：$P(S_1 \le s \mid N(t)=1) = s/t$
5.  **Kolmogorov定理核心**：相容的有限维分布族 $\{F_{t_1, \ldots, t_n}\}$ $\Rightarrow$ 存在唯一的随机过程 $\{X(t)\}$。

## 易混淆点

- **泊松分布 vs. 指数分布**：泊松分布描述的是**固定时间段内事件发生的次数**，是离散分布；指数分布描述的是**连续两个事件之间的时间间隔**，是连续分布。两者是描述泊松过程不同侧面的“孪生”分布。
- **Kolmogorov定理的应用**：该定理是**存在性**定理，它保证了在给定相容的有限维分布后，相应的随机过程在数学上是“存在”的。但它并不提供构造该过程的具体方法。
- **条件时刻的均匀性**：$S_1$ 在 $N(t)=1$ 条件下服从均匀分布是一个**条件分布**的性质，并非 $S_1$ 本身的无条件分布。$S_1$ 本身（首次到达时间）服从指数分布。
