# 泊松过程核心理论：分布、性质与Kolmogorov存在性定理

## 核心概念定义

**泊松过程**是一种重要的计数过程，用于描述在单位时间或单位空间内，某随机事件发生的次数。它是[[泊松分布]]在连续时间上的推广。

**Kolmogorov定理**（此处主要指Kolmogorov存在性定理或一致性定理）是[[概率论]]与[[随机过程]]中的基本定理，它给出了一个随机过程存在的充分必要条件，即其有限维分布族需满足对称性与相容性。

## 详细说明

### 1. 泊松分布及其极限形式
泊松分布是描述稀有事件发生次数的经典离散分布。其概率质量函数为：
$$P(N(t)=k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}$$
其中，$\lambda$ 是事件发生的强度（平均速率），$t$ 是时间长度。

该分布可由二项分布的极限推导而来：
$$\lim_{n\to\infty} C_n^k p^k (1-p)^{n-k} = \frac{(\lambda t)^k}{k!} e^{-\lambda t}$$
条件是 $np \to \lambda t$ 当 $n \to \infty$。==泊松分布的极限形式是连接离散与连续模型的关键。==

### 2. Kolmogorov存在性定理
该定理是构建随机过程的基石。
- **条件**：设有一族有限维分布函数 $F_X(x_1, \cdots, x_n; t_1, \cdots, t_n),\; t_1, \cdots, t_n \in T$，若其满足：
  1. **对称性**：对 $(t_1, ..., t_n)$ 的任意排列，分布函数值不变。
  2. **相容性**：低维分布可由高维分布通过边缘化得到。
- **结论**：则必存在一个概率空间及其上的随机过程 $\{X(t), t \in T\}$，使得该过程的所有有限维分布恰好是给定的分布族。
==Kolmogorov定理确保了满足对称性与相容性的分布族能唯一对应一个随机过程。==

### 3. 泊松过程的性质
#### 协方差性质
泊松过程是独立增量过程，也是一阶齐次过程。记 $N(t)$ 为强度 $\lambda$ 的泊松过程，其协方差为：
$$\mathrm{cov}(N(s), N(t)) = \lambda \min(s, t)$$
推导利用了独立增量性：$\mathrm{cov}(N(s), N(t)) = \mathrm{cov}(N(s), N(s) + (N(t)-N(s))) = \mathrm{Var}(N(s)) = \lambda s$（当 $s \le t$）。

#### 可加性（叠加性）
两个独立的泊松过程之和仍为泊松过程，强度为两者之和。
设 $N_1(t) \sim P(\lambda_1 t),\, N_2(t) \sim P(\lambda_2 t)$ 且独立，则：
$$N(t) = N_1(t) + N_2(t) \sim P((\lambda_1+\lambda_2)t)$$
==泊松分布的可加性是泊松过程叠加性质的基础。==

#### 事件发生时刻的条件分布
这是一个非常重要的性质。考虑一个强度为 $\lambda$ 的泊松过程，已知在 $[0, t]$ 内只发生了一次事件，记该事件的发生时刻为 $S_1$。则 $S_1$ 在 $[0, t]$ 上服从[[均匀分布]]：
$$P(S_1 \leq s \mid N(t) = 1) = \frac{s}{t}, \quad 0 \le s \le t$$
推导过程：
$$P(S_1 \leq s \mid N(t) = 1) = \frac{P(N(s)=1, N(t)-N(s)=0)}{P(N(t)=1)} = \frac{(\lambda s e^{-\lambda s}) \cdot e^{-\lambda (t-s)}}{\lambda t e^{-\lambda t}} = \frac{s}{t}$$
这表明，在已知仅发生一次事件的条件下，该事件的发生时刻是“随机地”落在区间内的。

### 4. 联合分布与阶秩统计量
在分析多个事件发生时刻 $S_1, S_2, ..., S_n$（已知 $N(t)=n$）时，会涉及到**阶秩统计量**。
- 它们的联合概率密度函数为：
$$f_{S_1, ..., S_n}(s_1, ..., s_n \mid N(t)=n) = \frac{n!}{t^n}, \quad 0 < s_1 < s_2 < ... < s_n < t$$
这里的 $n!$ 因子体现了 $n$ 个事件发生时间的所有可能排列。
- 更一般地，对于从某个分布 $F$ 中抽取的样本，其阶秩统计量 $Y_{(1)}, ..., Y_{(n)}$ 的联合密度也包含 $n!$ 因子。

## 关键公式
1. **泊松分布**：$P(N(t)=k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}$
2. **协方差**：$\mathrm{cov}(N(s), N(t)) = \lambda \min(s, t)$
3. **可加性**：$N_1(t) + N_2(t) \sim P((\lambda_1+\lambda_2)t)$ （独立时）
4. **单事件时刻条件分布**：$P(S_1 \leq s \mid N(t) = 1) = s/t$
5. **多事件时刻联合密度（条件于N(t)=n）**：$f(s_1,...,s_n) = n!/t^n, \, 0<s_1<...<s_n<t$

## 关联知识
- **基础**：[[概率论]]， [[概率分布]]， [[离散型随机变量]]， [[连续型随机变量]]， [[均匀分布]]， [[指数分布]]（与泊松过程的事件间隔时间相关）。
- **进阶**：[[Kolmogorov 一致性定理]]， [[不可约马尔可夫链]]（同为随机过程重要模型）， [[测度论]]与[[实变函数]]（为高级概率论提供理论基础）。
- **应用**：排队论、可靠性工程、金融数学中的跳跃过程。

## 易混淆点
- **泊松过程 vs. 泊松分布**：泊松分布描述固定时间段内事件数的分布，是离散的；泊松过程描述事件随时间发生的整个动态，是连续的随机过程。后者包含前者作为其有限维分布的一部分。
- **Kolmogorov定理的应用**：该定理是存在性定理，告诉我们什么样的分布族可以定义一个过程，但它并不提供构造过程的具体方法。
- **条件时刻的均匀分布**：该性质仅在**已知事件发生次数**的条件下成立。无条件时，泊松过程的事件发生时刻本身并不独立或均匀。
- **阶秩统计量中的 $n!$**：这个因子来源于所有可能的顺序排列。在计算涉及有序统计量的概率时，必须考虑所有排列，否则会遗漏概率质量。
