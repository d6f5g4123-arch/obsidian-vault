# 泊松过程与计数过程：定义、性质、等价形式与关键定理

# 泊松过程与计数过程

## 核心概念定义

- **[[随机过程]]**：设 $T$ 是 $[0, +\infty)$ 上某非空集合，$\forall t \in T$，$X_t$ 是随机变量，称 $\{X_t \mid t \in T\}$ 为随机过程。
- **[[计数过程]]**：记作 $\{N(t)\}$，用于计量给定时间区间内事件发生的总次数。其核心属性为：
  1. **独立增量性**：任意不相交时间区间内的事件数相互独立。
  2. **齐性/平稳增量性**：增量的概率分布仅依赖于区间长度，与起点无关。
- **[[泊松过程]]**：最常见的计数过程，是“定比分过程”。

## 泊松过程的等价定义

泊松过程有两种等价的定义方式。

### 定义一（基于分布）
一个计数过程 $\{N(t), t \ge 0\}$ 称为强度为 $\lambda$ 的泊松过程，如果满足：
1. $N(0) = 0$。
2. 具有独立增量性与齐性增量性。
3. 对任意 $s, t \ge 0$，增量 $N(s, s+t) = N(s+t) - N(s)$ 服从参数为 $\lambda t$ 的泊松分布：
   $$P(N(s, s+t) = k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}$$

### 定义二（基于小区间概率）
一个计数过程 $\{N(t), t \ge 0\}$ 称为强度为 $\lambda$ 的泊松过程，如果满足：
1. $N(0) = 0$。
2. $N(t)$ 非减且右连续。
3. 对任意 $t \ge 0$，当 $h \to 0^+$ 时，有：
   $$\begin{aligned}
   P(N(h) = 1) &= \lambda h + o(h) \\
   P(N(h) \ge 2) &= o(h)
   \end{aligned}$$
   （隐含 $P(N(h) = 0) = 1 - \lambda h + o(h)$）

==**泊松过程定义与独立/齐性增量性**是最基本考试高频考点。==
==**小区间概率和两个定义的等价性证明**是理解泊松过程的理论关键。==

## 详细性质与定理

### 1. 泊松过程的基本性质
- **泊松分布的极限定理**：泊松分布可由二项分布的极限推导得出。
- **分割区间法证明等价性**：将区间 $[0, t]$ 等分为 $n$ 段，记 $t_j = j \frac{t}{n}$，$Y_j = N(t_{j-1}, t_j)$。当 $n \to \infty$ 时，有：
  $$\begin{aligned}
  P(Y_j = 1) &= \frac{\lambda t}{n} + o\left(\frac{1}{n}\right) \\
  P(Y_j = 0) &= 1 - \frac{\lambda t}{n} - o\left(\frac{1}{n}\right) \\
  P(Y_j \ge 2) &= o\left(\frac{1}{n}\right)
  \end{aligned}$$
  则“有 $k$ 个 $Y_j=1$，其余为0”的概率极限即为泊松分布：
  $$\lim_{n\to\infty} C_n^k p^k q^{n-k} = \frac{(\lambda t)^k}{k!} e^{-\lambda t}$$

### 2. 协方差性质
泊松过程是一阶齐次过程。对于 $0 \le s \le t$，其协方差为：
$$\mathrm{cov}(N(s), N(t)) = \lambda s$$

### 3. 独立泊松过程的可加性
若 $N_1(t) \sim \mathrm{Poisson}(\lambda_1 t)$，$N_2(t) \sim \mathrm{Poisson}(\lambda_2 t)$ 相互独立，则它们的和仍为泊松过程：
$$N_1(t) + N_2(t) \sim \mathrm{Poisson}((\lambda_1+\lambda_2)t)$$
**推导**：
$$\begin{aligned}
P(N(t) = n) &= \sum_{k=0}^n P(N_1(t) = k)P(N_2(t) = n-k) \\
&= \sum_{k=0}^n \frac{(\lambda_1 t)^k}{k!}e^{-\lambda_1 t} \cdot \frac{(\lambda_2 t)^{n-k}}{(n-k)!}e^{-\lambda_2 t} \\
&= e^{-(\lambda_1+\lambda_2)t} \frac{[ (\lambda_1 + \lambda_2)t ]^n}{n!}
\end{aligned}$$
==**泊松分布的极限形式、可加性**是典型考点。==

### 4. 事件发生时刻的条件分布（阶秩统计量）
- **单事件情形**：若已知在 $[0, t]$ 内仅发生一次事件（即 $N(t) = 1$），记 $S_1$ 为该事件发生时刻，则 $S_1$ 在 $[0, t]$ 上服从**[[均匀分布]]**：
  $$P(S_1 \leq s \mid N(t) = 1) = \frac{s}{t}$$
- **多事件情形**：更一般地，已知在长度为 $T$ 的区间内发生了 $n$ 个事件，则这 $n$ 个事件的发生时刻 $Y_1, \ldots, Y_n$ 的联合概率密度函数为：
  $$f(y_1, \ldots, y_n) = n! \cdot \frac{1}{T^n}$$
  这相当于 $n$ 个独立同分布于 $[0, T]$ 上的均匀随机变量的**[[阶秩统计量]]**。
==**单事件释放时刻的均匀性**体现概率分布直觉。==
==**阶秩统计量密度**也是典型考点。==

## 关键公式总结
1. **泊松分布概率公式**：
   $$P(N(s, s+t) = k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}$$
2. **小区间概率（DEF12）**：
   $$\begin{aligned}
   P(N(h) = 1) &= \lambda h + o(h) \\
   P(N(h) \ge 2) &= o(h)
   \end{aligned}$$
3. **协方差公式**：
   $$\mathrm{cov}(N(s), N(t)) = \lambda s$$
4. **独立泊松过程和的分布**：
   $$P(N_1(t)+N_2(t) = n) = \frac{[ (\lambda_1 + \lambda_2)t ]^n}{n!} e^{-(\lambda_1 + \lambda_2)t}$$
5. **单事件时刻条件分布**：
   $$P(S_1 \leq s \mid N(t) = 1) = \frac{s}{t}$$
6. **多事件时刻联合密度**：
   $$f(y_1, \ldots, y_n) = n! \cdot \frac{1}{T^n}$$

## 易混淆点
- **独立增量 vs 平稳增量**：独立增量强调不同区间事件的独立性；平稳增量强调增量分布只依赖于区间长度。泊松过程同时具备这两个性质。
- **泊松过程的两种定义**：基于整体分布的定义（定义一）更直观，基于小区间概率的定义（定义二）更便于理论推导和推广（如非齐次泊松过程）。两者等价性的证明是理解泊松过程本质的关键。
