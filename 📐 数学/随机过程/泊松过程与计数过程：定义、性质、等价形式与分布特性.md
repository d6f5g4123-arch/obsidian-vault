# 泊松过程与计数过程：定义、性质、等价形式与分布特性

# 核心概念定义

- **[[随机过程]]**：设 $T$ 是 $[0, +\infty)$ 上的非空集合，对任意 $t \in T$，$X_t$ 是一个随机变量，则称 $\{X_t \mid t \in T\}$ 为一个随机过程。
- **[[计数过程]]（Counting Process）**：记作 $\{N(t)\}$，用于计量在给定时间区间内事件发生的总次数。其核心属性包括：
  1. **独立增量性（Independent Increment）**：任意不相交时间区间内发生的事件数相互独立。
  2. **齐性/平稳增量性（Stationary Increment）**：增量的概率分布只与区间长度有关，与起点无关。
- **[[泊松过程]]（Poisson Process）**：一种最常见的计数过程，是“定比分过程”。其标准定义如下：
  1. $N(0) = 0$。
  2. 具有独立增量性与齐性增量性。
  3. 对任意 $s, t \ge 0$，增量 $N(s, s+t) = N(s+t) - N(s)$ 服从参数为 $\lambda t$ 的泊松分布，即：
     $$P(N(s, s+t) = k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}$$

## 泊松过程的等价定义（DEF12形式）

泊松过程也可以由以下“小步概率”性质等价定义：
1. $N(0) = 0$。
2. $N(t)$ 非减且右连续。
3. 对任意 $t \ge 0$，当 $h \to 0^+$ 时，有：
   $$\begin{aligned}
   P(N(h) = 1) &= \lambda h + o(h) \\
   P(N(h) \ge 2) &= o(h)
   \end{aligned}$$
   其中 $o(h)$ 表示 $h$ 的高阶无穷小。

==**泊松过程定义与独立/齐性增量性**是最基本且高频的考点。==

# 详细性质与证明

## 泊松过程的基本性质

- **独立增量性**和**齐性增量性**是判断一个计数过程是否为泊松过程的两个基本条件。
- **泊松分布的极限定理**：泊松分布的概率质量函数可由二项分布的极限推导得出。
- ==**两个定义的等价性证明**是理解泊松过程的理论关键。== 其核心思想是**分割区间法**：
  将区间 $[0, t]$ 等分为 $n$ 段，记 $t_j = j \frac{t}{n}$，$Y_j = N(t_{j-1}, t_j)$。在 DEF12 定义下，当 $n$ 很大时，有：
  $$\begin{aligned}
  P(Y_j = 1) &= \frac{\lambda t}{n} + o\left(\frac{1}{n}\right) \\
  P(Y_j = 0) &= 1 - \frac{\lambda t}{n} - o\left(\frac{1}{n}\right) \\
  P(Y_j \ge 2) &= o\left(\frac{1}{n}\right)
  \end{aligned}$$
  考虑“恰好有 $k$ 个 $Y_j=1$，其余为0”这一事件的概率，当 $n \to \infty$ 时，其极限即为泊松分布的概率 $\frac{(\lambda t)^k}{k!} e^{-\lambda t}$。

## 协方差性质

泊松过程是一阶齐次过程。其协方差公式为：
$$\mathrm{cov}(N(s), N(t)) = \lambda \min(s, t)$$
（注：原始笔记中 $\lambda s$ 应为 $\lambda \min(s, t)$ 的特例，当 $s \le t$ 时成立。）

## 泊松过程的可加性

若 $N_1(t) \sim \mathrm{Poisson}(\lambda_1 t)$ 与 $N_2(t) \sim \mathrm{Poisson}(\lambda_2 t)$ 相互独立，则它们的和仍是一个泊松过程：
$$N_1(t) + N_2(t) \sim \mathrm{Poisson}((\lambda_1+\lambda_2)t)$$
**推导**：
$$\begin{aligned}
P(N(t) = n) &= \sum_{k=0}^n P(N_1(t) = k)P(N_2(t) = n-k) \\
&= \sum_{k=0}^n \frac{(\lambda_1 t)^k}{k!}e^{-\lambda_1 t} \cdot \frac{(\lambda_2 t)^{n-k}}{(n-k)!}e^{-\lambda_2 t} \\
&= e^{-(\lambda_1+\lambda_2)t} \frac{[(\lambda_1 + \lambda_2)t]^n}{n!}
\end{aligned}$$
==**泊松分布的可加性**是典型考点。==

## 事件发生时刻的条件分布

这一性质揭示了泊松过程事件发生时刻的均匀性。
- **单事件情况**：已知在 $[0, t]$ 内仅发生一次事件（即 $N(t) = 1$），记 $S_1$ 为该事件的发生时刻，则 $S_1$ 在 $[0, t]$ 上服从**[[均匀分布]]**：
  $$P(S_1 \leq s \mid N(t) = 1) = \frac{s}{t}, \quad 0 \le s \le t$$
- **多事件情况**：更一般地，已知在长度为 $T$ 的区间内发生了 $n$ 个事件，则这 $n$ 个事件的发生时刻 $Y_1, \ldots, Y_n$ 的联合概率密度函数为：
  $$f(y_1, \ldots, y_n) = n! \cdot \frac{1}{T^n}, \quad 0 < y_1 < \ldots < y_n < T$$
  这恰好是 $n$ 个独立同分布于 $U(0, T)$ 的随机变量的**[[阶秩统计量]]**的联合密度。
==**单事件释放时刻的均匀性**体现了泊松过程的无记忆性和随机性，是重要的概率直觉和考点。==

# 关键公式总结

1. **泊松分布概率公式**
   $$P(N(s, s+t) = k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}$$
2. **小步概率定义（DEF12）**
   $$\begin{aligned}
   P(N(h) = 1) &= \lambda h + o(h) \\
   P(N(h) \ge 2) &= o(h)
   \end{aligned}$$
3. **协方差公式**
   $$\mathrm{cov}(N(s), N(t)) = \lambda \min(s, t)$$
4. **独立泊松过程和的分布**
   $$P(N_1(t)+N_2(t) = n) = \frac{[(\lambda_1 + \lambda_2)t]^n}{n!} e^{-(\lambda_1 + \lambda_2)t}$$
5. **事件发生时刻的条件分布**
   - 单事件：$P(S_1 \leq s \mid N(t) = 1) = \frac{s}{t}$
   - $n$个事件：$f(y_1, \ldots, y_n) = n! \cdot \frac{1}{T^n}$

# 易混淆点

- **独立增量 vs. 平稳增量**：独立增量强调不同区间事件的独立性；平稳增量强调增量分布仅依赖于区间长度。泊松过程同时具备这两者。
- **泊松过程的两种定义**：一种基于增量的泊松分布（标准定义），另一种基于无穷小区间内的概率行为（DEF12）。理解它们之间的等价性是关键。
- **事件发生时刻的分布**：无条件时，泊松过程的事件发生间隔服从指数分布；但在给定事件总数条件下，事件发生时刻在区间内是均匀分布的（阶秩统计量）。这是两个不同但相关的结论。
