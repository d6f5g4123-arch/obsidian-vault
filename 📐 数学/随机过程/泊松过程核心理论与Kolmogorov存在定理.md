# 泊松过程核心理论与Kolmogorov存在定理

## 核心概念定义

**泊松过程**是一种重要的**随机过程**，用于描述单位时间或单位空间内某事件发生次数的随机现象。其核心是**泊松分布**。

**Kolmogorov定理**（此处特指Kolmogorov一致性定理或存在性定理）是**概率论**与**随机过程**的基石，它给出了一个随机过程存在的充分必要条件。

## 详细说明

### 1. 泊松分布

**泊松分布**是一种描述单位时间或单位空间内某事件发生次数的离散概率分布。

- **概率质量函数**：
  $$P(N(t)=k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}$$
  其中，$\lambda > 0$ 是事件发生的平均速率，$t$ 是时间长度，$N(t)$ 表示在 $[0, t]$ 时间内事件发生的次数。

- **极限形式**：泊松分布可以看作是二项分布在 $n \to \infty, p \to 0, np \to \lambda t$ 时的极限。
  $$\lim_{n\to\infty} C_n^k p^k (1-p)^{n-k} = \frac{(\lambda t)^k}{k!} e^{-\lambda t}$$

### 2. 泊松过程的关键性质

- **独立增量性**：在不相交的时间区间内，事件发生的次数是相互独立的。
- **平稳增量性**：在任意长度相同的时间区间内，事件发生次数的概率分布只依赖于区间的长度，而与起点无关。
- **普通性**：在极短的时间内，发生两次或以上事件的概率是 $o(\Delta t)$。

### 3. Kolmogorov存在定理

该定理是构建随机过程的数学基础。

- **内容**：设给定一族有限维分布函数 $F_X(x_1, \cdots, x_n; t_1, \cdots, t_n),\; t_1, \cdots, t_n \in T$。如果这族分布函数满足**对称性**和**相容性**，则必存在一个概率空间 $ (\Omega, \mathcal{F}, P) $ 及其上的随机过程 $X(t)$，使得该过程的所有有限维分布恰好等于给定的分布族。即：
  $$F_X(x_1, \cdots, x_n; t_1, \cdots, t_n) = P(X(t_1) \le x_1, \cdots, X(t_n) \le x_n)$$
- **重点**：==该定理是随机过程存在的充分必要条件==，它保证了只要有限维分布相容，整个随机过程在数学上就是良定义的。

### 4. 泊松过程的数字特征

- **均值**：$E[N(t)] = \lambda t$
- **方差**：$Var[N(t)] = \lambda t$
- **协方差**：对于 $0 \le s \le t$，有
  $$\mathrm{cov}(N(s), N(t)) = \mathrm{cov}(N(s), N(t)-N(s) + N(s)) = \mathrm{cov}(N(s), N(s)) = \lambda s$$
  这表明泊松过程是二阶矩过程。

### 5. 泊松过程的到达时刻分布

设 $S_1$ 为泊松过程第一次事件发生的时刻（首次到达时刻）。在已知 $[0, t]$ 内仅发生一次事件（即 $N(t)=1$）的条件下，该事件的发生时刻服从均匀分布。

- **推导**：
  $$P(S_1 \leq s \mid N(t) = 1) = \frac{P(N(s) = 1, N(t) - N(s) = 0)}{P(N(t) = 1)} = \frac{[\lambda s e^{-\lambda s}] \cdot e^{-\lambda (t-s)}}{\lambda t \cdot e^{-\lambda t}} = \frac{s}{t}$$
- **结论**：==在已知仅发生一次事件的条件下，该事件的发生时刻在 $[0, t]$ 上服从均匀分布==。

### 6. 泊松分布的可加性

若 $N_1(t) \sim P(\lambda_1 t)$ 与 $N_2(t) \sim P(\lambda_2 t)$ 是**相互独立**的泊松过程，则它们的和仍服从泊松分布：
$$N_1(t) + N_2(t) \sim P((\lambda_1+\lambda_2)t)$$

**推导**：
$$P(N_1(t)+N_2(t)=n) = \sum_{k=0}^n P(N_1(t)=k) P(N_2(t)=n-k) = \frac{[ (\lambda_1+\lambda_2)t ]^n}{n!} e^{-(\lambda_1+\lambda_2)t}$$
==重点：独立泊松过程之和仍为泊松过程，其强度为原强度之和。==

## 关键公式

1. **泊松分布公式**：
   $$P(N(t)=k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}$$
2. **Kolmogorov定理**：有限维分布族 $\{F_X(\cdot; t_1, ..., t_n)\}$ 满足对称性、相容性 $\Rightarrow$ 存在唯一随机过程 $X(t)$。
3. **泊松过程协方差**：$\mathrm{cov}(N(s), N(t)) = \lambda \min(s, t)$。
4. **可加性**：$N_1(t)+N_2(t) \sim P((\lambda_1+\lambda_2)t)$。
5. **条件到达时刻分布**：$P(S_1 \leq s \mid N(t) = 1) = s/t$。

## 易混淆点

1. **泊松过程 vs. 泊松分布**：泊松分布描述的是固定时间区间内事件次数的**静态分布**；泊松过程则描述了事件随时间发生的**动态随机过程**，它由泊松分布刻画其增量。
2. **Kolmogorov定理的条件**：定理要求有限维分布族满足严格的**对称性**（对指标置换不变）和**相容性**（高维分布边缘化后与低维分布一致），这是过程存在的关键，而非任意给定的分布都能构成一个过程。
3. **到达时刻的均匀分布**：该性质仅在**已知区间内事件总数为1**的条件下成立。一般情况下，多个到达时刻的联合分布是[[阶秩统计量]]的分布，而非独立均匀分布。
4. **可加性的前提**：必须是**相互独立**的泊松过程之和才保持泊松性。如果过程相关，则不成立。
