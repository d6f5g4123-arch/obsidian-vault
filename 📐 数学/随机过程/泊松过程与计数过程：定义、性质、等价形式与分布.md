# 泊松过程与计数过程：定义、性质、等价形式与分布

# 核心概念定义

- **[[随机过程]]**：设 $T$ 是 $[0, +\infty)$ 上某非空集合，$\forall t \in T$，$X_t$ 是随机变量，则 $\{X_t \mid t \in T\}$ 称为一个随机过程。
- **[[计数过程]]**：记作 $\{N(t)\}$，用于计量给定时间区间内事件发生的总次数。其核心属性包括：
  - **独立增量性**：任意不相交时间区间内发生的事件数相互独立。
  - **齐性/平稳增量性**：增量的概率分布只与区间长度有关，与起点无关。
- **[[泊松过程]]**：一种最常见的计数过程，具有两种等价的定义形式。

## 泊松过程的等价定义

### 定义一（基于分布）
满足以下三个条件的计数过程 $\{N(t), t \ge 0\}$ 称为强度为 $\lambda$ 的泊松过程：
1. $N(0) = 0$。
2. 具有独立增量性与齐性增量性。
3. 对任意 $s, t \ge 0$，增量 $N(s, s+t) \sim \mathrm{Poisson}(\lambda t)$，即
   $$P(N(s, s+t) = k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}$$

### 定义二（基于小区间概率）
满足以下三个条件的计数过程 $\{N(t), t \ge 0\}$ 称为强度为 $\lambda$ 的泊松过程：
1. $N(0) = 0$。
2. $N(t)$ 是非减且右连续的。
3. 对任意 $t \ge 0$，当 $h \to 0^+$ 时，有
   $$\begin{aligned}
   P(N(t, t+h) = 1) &= \lambda h + o(h) \\
   P(N(t, t+h) \ge 2) &= o(h)
   \end{aligned}$$
   其中 $o(h)$ 表示 $h$ 的高阶无穷小。

==**泊松过程定义与独立/齐性增量性**是最基本考试高频考点。==
==**小区间概率和两个定义的等价性证明**是理解泊松过程的理论关键。==

# 详细性质与证明

## 基本性质
- **泊松分布的极限定理**：泊松分布可由二项分布的极限推导而来。
- **分割区间法证明等价性**：通过将区间 $[0, t]$ 细分为 $n$ 段，可以证明定义二能推出定义一中的泊松分布形式。
  设 $t_j = j \frac{t}{n}$，$Y_j = N(t_{j-1}, t_j)$，则有：
  $$\begin{aligned}
  P(Y_j = 1) &= \frac{\lambda t}{n} + o\left(\frac{1}{n}\right) \\
  P(Y_j = 0) &= 1 - \frac{\lambda t}{n} - o\left(\frac{1}{n}\right) \\
  P(Y_j \ge 2) &= o\left(\frac{1}{n}\right)
  \end{aligned}$$
  当 $n \to \infty$ 时，“有 $k$ 个 $Y_j=1$，其余为0”的概率极限即为泊松分布：
  $$\lim_{n\to\infty} C_n^k p^k q^{n-k} = \frac{(\lambda t)^k}{k!} e^{-\lambda t}$$

## 协方差性质
泊松过程是一阶齐次过程。其协方差公式为：
$$\mathrm{cov}(N(s), N(t)) = \lambda \min(s, t)$$
（注：原始笔记中 $\mathrm{cov}(N(s), N(t)) = \lambda s$ 应为 $s \le t$ 时的特例。）

## 可加性
若 $N_1(t) \sim \mathrm{Poisson}(\lambda_1 t)$ 与 $N_2(t) \sim \mathrm{Poisson}(\lambda_2 t)$ 相互独立，则它们的和仍为泊松过程：
$$N_1(t) + N_2(t) \sim \mathrm{Poisson}((\lambda_1+\lambda_2)t)$$
推导过程：
$$\begin{aligned}
P(N(t) = n) &= \sum_{k=0}^n P(N_1(t) = k)P(N_2(t) = n-k) \\
&= \sum_{k=0}^n \frac{(\lambda_1 t)^k}{k!}e^{-\lambda_1 t} \cdot \frac{(\lambda_2 t)^{n-k}}{(n-k)!}e^{-\lambda_2 t} \\
&= e^{-(\lambda_1+\lambda_2)t} \frac{[ (\lambda_1 + \lambda_2)t ]^n}{n!}
\end{aligned}$$
==**泊松分布的极限形式、可加性**是典型考点。==

# 事件发生时刻的分布

## 单事件情形
已知在 $[0, t]$ 内仅发生一次事件（即 $N(t) = 1$），记 $S_1$ 为该事件的发生时刻，则 $S_1$ 在 $[0, t]$ 上服从**[[均匀分布]]**：
$$P(S_1 \leq s \mid N(t) = 1) = \frac{s}{t}, \quad 0 \le s \le t$$
==**单事件释放时刻的均匀性**体现概率分布直觉。==

## 多事件情形（阶秩统计量）
更一般地，已知在长度为 $T$ 的区间内发生了 $n$ 个事件，则这 $n$ 个事件的发生时刻 $S_1, S_2, \ldots, S_n$（已排序）的联合概率密度函数为：
$$f(y_1, \ldots, y_n) = n! \cdot \frac{1}{T^n}, \quad 0 < y_1 < y_2 < \ldots < y_n < T$$
这等价于 $n$ 个独立同分布的 $U(0, T)$ 随机变量的**[[阶秩统计量]]**的联合密度。
==**阶秩统计量密度**也是典型考点。==

# 关键公式汇总
1. **泊松分布概率**：$P(N(s, s+t) = k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}$
2. **小区间概率（定义二）**：$P(N(h)=1)=\lambda h+o(h), \quad P(N(h)\ge2)=o(h)$
3. **协方差**：$\mathrm{cov}(N(s), N(t)) = \lambda \min(s, t)$
4. **独立泊松过程之和**：$N_1(t)+N_2(t) \sim \mathrm{Poisson}((\lambda_1+\lambda_2)t)$
5. **单事件时刻分布**：$P(S_1 \leq s \mid N(t) = 1) = \frac{s}{t}$
6. **多事件时刻联合密度**：$f(y_1, \ldots, y_n) = n! \cdot \frac{1}{T^n}$
