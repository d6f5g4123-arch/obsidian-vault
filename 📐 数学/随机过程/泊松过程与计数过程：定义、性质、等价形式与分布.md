# 泊松过程与计数过程：定义、性质、等价形式与分布

# 核心概念定义

- **[[随机过程]]**：设 $T$ 是 $[0, +\infty)$ 上某非空集合，$\forall t \in T$，$X_t$ 是随机变量，则 $\{X_t \mid t \in T\}$ 称为随机过程。
- **[[计数过程]]**：记作 $\{N(t)\}$，用于计量在给定时间区间内事件发生的总次数。其核心属性为：
  - **独立增量性**：任意不相交时间区间内发生的事件数相互独立。
  - **齐性/平稳增量性**：增量的分布只与区间长度有关，与起点无关。
- **[[泊松过程]]**：一种最常见的计数过程，具有严格的数学定义。

# 泊松过程的定义与等价形式

泊松过程有两种等价的定义方式。

## 定义一（基于增量分布）
满足以下条件的计数过程 $\{N(t), t \ge 0\}$ 称为**强度为 $\lambda$ 的泊松过程**：
1. $N(0) = 0$。
2. 具有**独立增量性**与**齐性增量性**。
3. 对任意 $s, t \ge 0$，增量 $N(s, s+t) = N(s+t) - N(s)$ 服从泊松分布：
   $$N(s, s+t) \sim \mathrm{Poisson}(\lambda t)$$
   即
   $$P(N(s, s+t) = k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}, \quad k = 0, 1, 2, \ldots$$

## 定义二（基于小区间概率）
满足以下条件的计数过程 $\{N(t), t \ge 0\}$ 称为**强度为 $\lambda$ 的泊松过程**：
1. $N(0) = 0$。
2. $N(t)$ 是非负整数、非减且右连续的。
3. 对任意 $t \ge 0$，当 $h \to 0^+$ 时，有
   $$
   \begin{aligned}
   P(N(t, t+h) = 1) &= \lambda h + o(h) \\
   P(N(t, t+h) \ge 2) &= o(h)
   \end{aligned}
   $$
   其中 $o(h)$ 表示 $h$ 的高阶无穷小。

==**两种定义的等价性是理解泊松过程的理论关键，也是高频考点。**==

# 泊松过程的基本性质

## 1. 泊松分布的极限定理
泊松分布可由二项分布的极限推导而来，这为泊松过程的定义提供了直观背景。

## 2. 分割区间法证明等价性
通过将区间 $[0, t]$ 细分为 $n$ 段，可以证明定义二能推导出定义一中的泊松分布形式。设 $t_j = j t / n$，$Y_j = N(t_{j-1}, t_j)$，则有：
$$
\begin{aligned}
P(Y_j = 1) &= \frac{\lambda t}{n} + o\left(\frac{1}{n}\right) \\
P(Y_j = 0) &= 1 - \frac{\lambda t}{n} - o\left(\frac{1}{n}\right) \\
P(Y_j \ge 2) &= o\left(\frac{1}{n}\right)
\end{aligned}
$$
当 $n \to \infty$ 时，“有 $k$ 个 $Y_j=1$，其余为0”这一事件的概率极限即为泊松概率质量函数：
$$\lim_{n\to\infty} C_n^k p^k q^{n-k} = \frac{(\lambda t)^k}{k!} e^{-\lambda t}$$

## 3. 协方差性质
泊松过程是一阶齐次过程，其协方差公式为：
$$\mathrm{cov}(N(s), N(t)) = \lambda \min(s, t)$$
特别地，当 $s \le t$ 时，$\mathrm{cov}(N(s), N(t)) = \lambda s$。

## 4. 独立泊松过程的可加性
若 $N_1(t) \sim \mathrm{Poisson}(\lambda_1 t)$ 与 $N_2(t) \sim \mathrm{Poisson}(\lambda_2 t)$ 相互独立，则它们的和仍为泊松过程：
$$N_1(t) + N_2(t) \sim \mathrm{Poisson}((\lambda_1+\lambda_2)t)$$
**推导**：
$$
\begin{aligned}
P(N(t) = n) &= \sum_{k=0}^n P(N_1(t) = k)P(N_2(t) = n-k) \\
&= \sum_{k=0}^n \frac{(\lambda_1 t)^k}{k!}e^{-\lambda_1 t} \cdot \frac{(\lambda_2 t)^{n-k}}{(n-k)!}e^{-\lambda_2 t} \\
&= e^{-(\lambda_1+\lambda_2)t} \frac{[(\lambda_1 + \lambda_2)t]^n}{n!}
\end{aligned}
$$
==**泊松分布的可加性是典型考点。**==

# 泊松过程中事件发生时刻的分布

## 1. 单事件发生时刻的条件分布
已知在 $[0, t]$ 内仅发生一次事件（即 $N(t)=1$），记 $S_1$ 为该事件的发生时刻，则 $S_1$ 在 $[0, t]$ 上服从**[[均匀分布]]**：
$$P(S_1 \leq s \mid N(t) = 1) = \frac{s}{t}, \quad 0 \le s \le t$$
==**这一性质体现了泊松过程事件发生的“完全随机性”，是重要的概率直觉。**==

## 2. 多事件发生时刻的联合分布（阶秩统计量）
更一般地，已知在长度为 $T$ 的区间内发生了 $n$ 个事件，则这 $n$ 个事件的发生时刻 $S_1, S_2, \ldots, S_n$ 的联合概率密度函数为：
$$f(y_1, \ldots, y_n) = n! \cdot \frac{1}{T^n}, \quad 0 < y_1 < y_2 < \ldots < y_n < T$$
这等价于说，这 $n$ 个时刻是区间 $[0, T]$ 上独立同分布均匀随机变量的**[[阶秩统计量]]**。

# 关键公式总结

1. **泊松分布概率公式**：
   $$P(N(s, s+t) = k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}$$
2. **小区间概率（定义二）**：
   $$
   \begin{aligned}
   P(N(h) = 1) &= \lambda h + o(h) \\
   P(N(h) \ge 2) &= o(h)
   \end{aligned}
   $$
3. **协方差公式**：
   $$\mathrm{cov}(N(s), N(t)) = \lambda \min(s, t)$$
4. **独立泊松过程之和**：
   $$P(N_1(t)+N_2(t) = n) = \frac{[(\lambda_1 + \lambda_2)t]^n}{n!} e^{-(\lambda_1 + \lambda_2)t}$$
5. **单事件时刻条件分布**：
   $$P(S_1 \leq s \mid N(t) = 1) = \frac{s}{t}$$
6. **多事件时刻联合密度**：
   $$f(y_1, \ldots, y_n) = n! \cdot \frac{1}{T^n}$$

# 易混淆点与重点

- ==**泊松过程的定义必须同时满足独立增量性和齐性增量性，两者缺一不可。**== 这是最基本的判断依据。
- 区分“泊松分布”（描述固定区间内事件数的分布）和“泊松过程”（描述事件随时间发生的整个动态）。
- 理解定义二中的 $o(h)$ 项，它保证了在无穷小时间内几乎不可能发生两个或以上事件。
- 事件发生时刻的条件均匀分布特性，仅在**已知事件发生总数**的条件下成立。
