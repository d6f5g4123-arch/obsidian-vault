# 泊松过程及其相关分布与Kolmogorov定理

# 核心概念定义

**泊松过程（Poisson Process）** 是一种重要的**计数过程**，用于描述在单位时间或单位空间内，某个事件随机发生的次数。其核心特征是**平稳增量性**与**独立增量性**。

**Kolmogorov 定理** 是一组以苏联数学家安德雷·柯尔莫哥洛夫命名的数学定理，在概率论与随机过程领域，特指 **Kolmogorov 一致性定理**，它给出了随机过程存在的充分必要条件。

# 详细说明

## 1. 泊松分布与极限形式
泊松分布是一种描述单位时间或单位空间内某事件发生次数的**离散概率分布**。
- 其概率质量函数为：
  $$
  P(N(t)=k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}
  $$
  其中，$\lambda$ 是事件发生的平均速率，$t$ 是时间长度。
- 泊松分布可由二项分布取极限得到，体现了**稀有事件**的统计规律。

## 2. Kolmogorov 一致性定理
该定理是构建随机过程的基石。
- **条件**：设有一族有限维分布函数 $F_X(x_1, \cdots, x_m; t_1, \cdots, t_n),\; t_1, \cdots, t_n \in T$，若其满足**对称性**与**相容性**。
- **结论**：则必存在且唯一的随机过程 $X(t)$，使得该族函数恰好为 $X(t)$ 的**所有有限维分布**，即：
  $$
  F_X(x_1, \cdots, x_n; t_1, \cdots, t_n) 
  = P(X(t_1) \le x_1, \cdots, X(t_n) \le x_n)
  $$
- **重点**：==该定理是随机过程存在的充分必要条件==。

## 3. 泊松过程的性质
### 3.1 协方差性质
泊松过程是一阶齐次过程。记 $N(t)$ 为泊松过程，其协方差为：
$$
\mathrm{cov}(N(s), N(t)) = \lambda \min(s, t)
$$
推导利用了独立增量性：$\mathrm{cov}(N(s), N(t)) = \mathrm{cov}(N(s), N(s)) = \lambda s$（当 $s \le t$）。

### 3.2 可加性
两个独立的泊松过程之和仍服从泊松分布。
- 设 $N_1(t) \sim P(\lambda_1 t),\quad N_2(t) \sim P(\lambda_2 t)$ 且独立，
- 则 $N_1(t) + N_2(t) \sim P[(\lambda_1+\lambda_2)t]$。
- ==泊松分布具有可加性==。

## 4. 泊松过程中的事件时刻分布
### 4.1 单粒子释放时刻的条件分布
考虑泊松过程 $N(t)$，已知在 $[0, t]$ 内仅发生一次事件（即 $N(t)=1$），记该事件的发生时刻为 $S_1$。
其条件分布函数为：
$$
P(S_1 \leq s \mid N(t) = 1) = \frac{s}{t}, \quad 0 \le s \le t
$$
推导过程利用了泊松过程的定义与条件概率：
$$
P(S_1 \leq s \mid N(t) = 1) = \frac{P(N(s)=1, N(t)-N(s)=0)}{P(N(t)=1)} = \frac{(\lambda s e^{-\lambda s}) \cdot e^{-\lambda (t-s)}}{\lambda t e^{-\lambda t}} = \frac{s}{t}
$$
**结论**：==在已知仅发生一次事件的条件下，该事件的发生时刻在区间 $[0, t]$ 上服从[[均匀分布]]==。

### 4.2 阶秩统计量的联合密度
对于泊松过程的事件发生时刻 $S_1, S_2, ..., S_n$（已知 $N(t)=n$），其联合概率密度函数与**阶秩统计量**的联合密度有关。
一般形式为：
$$
f_{S_1, ..., S_n}(s_1, ..., s_n \mid N(t)=n) = n! \cdot \frac{1}{t^n}, \quad 0 < s_1 < ... < s_n < t
$$
其中 $n!$ 因子体现了所有可能排列顺序的影响。

## 5. 联合分布函数的性质
设 $(X, Y)$ 的**联合分布函数**为 $F(x, y) = P(X \le x, Y \le y)$。
定义 $G(x, y) = P(X > x, Y \le y)$，则有关系：
$$
G(x, y) = P(Y \le y) - F(x, y)
$$
对其求导，可得密度函数关系：
$$
\frac{\partial^2 G}{\partial y \partial x} = -\frac{\partial^2 F}{\partial y \partial x}
$$
此关系可推广至高维情形，用于推导复杂事件的联合密度。

# 关键公式/代码

1. **泊松分布公式**：
   $$
   P(N(t)=k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}
   $$
2. **泊松过程协方差**：
   $$
   \mathrm{cov}(N(s), N(t)) = \lambda \min(s, t)
   $$
3. **独立泊松过程之和**：
   $$
   N_1(t)+N_2(t) \sim P((\lambda_1+\lambda_2)t)
   $$
4. **条件均匀分布（单事件）**：
   $$
   P(S_1 \leq s \mid N(t) = 1) = \frac{s}{t}
   $$

# 例题/应用

**例题**：设某路口车辆到达服从强度 $\lambda = 2$（辆/分钟）的泊松过程。求：
1. 3分钟内恰好到达5辆车的概率。
2. 已知3分钟内只到了1辆车，求该车在前1分钟内到达的概率。

**解**：
1. $N(3) \sim P(2*3)=P(6)$，故 $P(N(3)=5) = \frac{6^5}{5!}e^{-6}$。
2. 由条件分布性质，$P(S_1 \le 1 \mid N(3)=1) = \frac{1}{3}$。

# 易混淆点

1. **泊松分布 vs. 指数分布**：泊松分布描述**计数**（离散），指数分布描述**等待时间**（连续）。在泊松过程中，事件间隔时间服从指数分布。
2. **Kolmogorov定理的条件**：==对称性==（有限维分布对变量和指标置换不变）与==相容性==（高维分布边缘化后与低维分布一致）缺一不可。
3. **条件时刻分布**：已知 $N(t)=n$ 时，事件发生时刻 $S_1, ..., S_n$ 的联合分布**不等于** $n$ 个独立均匀分布的联合分布，而是与阶秩统计量相关。
