# 泊松过程、Kolmogorov定理与相关分布性质

# 核心概念定义

## 泊松过程 (Poisson Process)
**泊松过程**是一种重要的**计数过程**，用于描述在单位时间或单位空间内，某随机事件发生次数的模型。其核心特征是**平稳增量性**与**独立增量性**。

## Kolmogorov 定理
**Kolmogorov 定理**（此处特指**Kolmogorov 一致性定理**）是**随机过程**存在性理论中的核心定理。它指出：若给定一族满足**对称性**与**相容性**的有限维分布函数，则必存在一个随机过程，其有限维分布恰好等于给定的分布族。

## 相关分布
- **泊松分布 (Poisson Distribution)**：描述单位时间/空间内事件发生次数的**离散概率分布**。
- **均匀分布 (Uniform Distribution)**：在某个区间内所有取值出现的概率相等的概率分布。
- **指数分布 (Exponential Distribution)**：与泊松过程紧密相关，描述事件间隔时间的连续分布。

# 详细说明

## 1. 泊松过程的核心性质
- **定义**：记 $N(t)$ 为在时间区间 $[0, t]$ 内事件发生的次数，若 $N(t)$ 满足：
  1. $N(0) = 0$。
  2. 具有**独立增量性**。
  3. 具有**平稳增量性**，且增量服从泊松分布：$N(t+s)-N(s) \sim P(\lambda t)$。
- **参数**：$\lambda > 0$ 为**强度**，表示单位时间内事件发生的平均次数。

## 2. Kolmogorov 一致性定理详解
该定理为构造随机过程提供了理论基础。
- **条件**：设有一族有限维分布函数 $F_X(x_1, \cdots, x_n;\, t_1, \cdots, t_n),\; t_1, \cdots, t_n \in T$，满足：
  1. **对称性**：对 $(t_1, \ldots, t_n)$ 的任意排列，分布函数值不变。
  2. **相容性**：对任意 $m < n$，有 $F_X(x_1, \cdots, x_m, \infty, \cdots, \infty;\, t_1, \cdots, t_n) = F_X(x_1, \cdots, x_m;\, t_1, \cdots, t_m)$。
- **结论**：则存在概率空间 $(\Omega, \mathcal{F}, P)$ 及其上的随机过程 $\{X(t), t \in T\}$，使得该过程的所有有限维分布恰好是给定的分布族。即：
  $$
  F_X(x_1, \cdots, x_n;\, t_1, \cdots, t_n) = P(X(t_1) \le x_1, \cdots, X(t_n) \le x_n)
  $$
  ==该定理是随机过程存在的充分必要条件。==

## 3. 泊松过程的深入性质
### 3.1 可加性
两个**独立**的泊松过程之和仍为泊松过程，强度为两者之和。
设 $N_1(t) \sim P(\lambda_1 t),\, N_2(t) \sim P(\lambda_2 t)$ 且独立，则：
$$
N(t) = N_1(t) + N_2(t) \sim P((\lambda_1+\lambda_2)t)
$$

### 3.2 协方差性质
泊松过程是**二阶矩过程**。对于 $0 \le s \le t$，其协方差为：
$$
\mathrm{cov}(N(s), N(t)) = \mathrm{cov}(N(s), N(s) + (N(t)-N(s))) = \mathrm{Var}(N(s)) = \lambda s
$$
==泊松过程的协方差等于时间较小者的方差。==

### 3.3 事件到达时刻的条件分布
这是一个关键应用。已知在 $[0, t]$ 内仅发生一次事件（即 $N(t)=1$），记该事件的到达时刻为 $S_1$，则其条件分布为：
$$
P(S_1 \le s \mid N(t)=1) = \frac{s}{t}, \quad 0 \le s \le t
$$
**推导**：
$$
P(S_1 \le s \mid N(t)=1) = \frac{P(N(s)=1, N(t)-N(s)=0)}{P(N(t)=1)} = \frac{(\lambda s e^{-\lambda s}) \cdot e^{-\lambda (t-s)}}{\lambda t e^{-\lambda t}} = \frac{s}{t}
$$
==结论：在已知仅发生一次事件的条件下，该事件的到达时刻在 $[0, t]$ 上服从**均匀分布**。== 这揭示了泊松过程与均匀分布之间的深刻联系。

### 3.4 与阶秩统计量的联系
将 $n$ 个在 $[0, T]$ 上独立同分布（服从均匀分布）的随机变量按大小排序，得到**阶秩统计量** $Y_{(1)}, \cdots, Y_{(n)}$。其联合概率密度函数为：
$$
f(y_1, \ldots, y_n) = n! \cdot \frac{1}{T^n}, \quad 0 < y_1 < \cdots < y_n < T
$$
==这里的 $n!$ 因子源于 $n$ 个点的所有可能排列顺序。== 这一结论在推导泊松过程事件到达时刻的联合分布时非常重要。

# 关键公式

## 泊松分布概率质量函数
$$
P(N(t)=k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}, \quad k=0,1,2,\ldots
$$
==这是泊松过程增量的核心公式。==

## 泊松过程的协方差
$$
\mathrm{cov}(N(s), N(t)) = \lambda \min(s, t)
$$

## 独立泊松过程之和
$$
\text{若 } N_1(t) \sim P(\lambda_1 t),\, N_2(t) \sim P(\lambda_2 t) \text{ 独立，则 } N_1(t)+N_2(t) \sim P((\lambda_1+\lambda_2)t)
$$

## 条件均匀分布（单事件到达时刻）
$$
F_{S_1|N(t)=1}(s) = P(S_1 \le s \mid N(t)=1) = \frac{s}{t}
$$
对应的条件密度为 $f_{S_1|N(t)=1}(s) = 1/t$。

# 例题/应用

1.  **问题**：设某路口车辆到达服从强度 $\lambda=2$（辆/分钟）的泊松过程。求：(a) 3分钟内恰好到达5辆车的概率；(b) 已知3分钟内来了3辆车，求第一辆车在1分钟内到达的概率。
    **解**：
    (a) $N(3) \sim P(2 \times 3 = 6)$，故 $P(N(3)=5) = \frac{6^5}{5!}e^{-6}$。
    (b) 由条件分布结论，$P(S_1 \le 1 \mid N(3)=3) = \frac{1}{3}$。

2.  **问题**：电话总机接到的呼叫数服从泊松过程，$\lambda_A=3$次/小时，$\lambda_B=5$次/小时，且相互独立。求一小时内总呼叫数超过10次的概率近似值。
    **解**：总过程强度 $\lambda = \lambda_A + \lambda_B = 8$。总呼叫数 $N(1) \sim P(8)$。$P(N(1)>10) = 1 - \sum_{k=0}^{10} \frac{8^k}{k!}e^{-8}$，可通过查表或计算得。

# 易混淆点

1.  **泊松分布 vs. 泊松过程**：泊松分布描述的是一个**固定时间段内**事件发生次数的**分布**；泊松过程描述的是事件随时间发生的**动态随机过程**，其增量服从泊松分布。
2.  **Kolmogorov 定理的条件**：务必牢记**对称性**与**相容性**两个条件缺一不可，它们是有限维分布能成为某个随机过程分布的充要条件。
3.  **条件到达时刻的分布**：==$P(S_1 \le s \mid N(t)=1) = s/t$ 这个均匀分布结论，仅在已知**恰好发生一次事件**的条件下成立。== 如果已知发生 $n$ 次事件，则到达时刻的联合分布是 $n$ 个独立均匀分布变量的阶秩统计量。
4.  **协方差计算**：$\mathrm{cov}(N(s), N(t)) = \lambda \min(s, t)$，而非 $\lambda st$ 或 $\lambda \max(s, t)$。
