# 布朗运动的定义、性质、变换及其首中时与反正弦律

# 布朗运动

## 定义与性质

### 定义
若随机过程 $\{ X(t), t \ge 0 \}$ 满足以下四个条件，则称之为**布朗运动**（或**维纳过程**）：
1. $X(0) = 0$；
2. 具有**独立增量**；
3. 对任意 $s, t \ge 0$，增量 $X(t) - X(s) \sim N(0, \sigma^2 |t - s|)$；
4. 样本轨道 $X(t)$ 是 $t$ 的连续函数（以概率1）。

若方差参数 $\sigma^2 = 1$，则称为**标准布朗运动**，常记作 $\{ B(t), t \ge 0 \}$。

### 基本性质
布朗运动具有以下核心性质：
1. **独立增量性**：对任意 $0 \le t_1 < t_2 < \dots < t_n$，增量 $X(t_2)-X(t_1), \dots, X(t_n)-X(t_{n-1})$ 相互独立。
2. **平稳增量性**：增量 $X(t+s) - X(t)$ 的分布只依赖于时间差 $s$，与起始时刻 $t$ 无关。
3. **对称性**：$E[X(t)] = 0$。
4. **方差函数的连续性**：方差函数 $\sigma^2(t) = \operatorname{Var}(X(t))$ 是 $t$ 的连续函数（又称扩散函数）。

由独立增量性与平稳增量性可推导出方差函数的可加性：
$$\sigma^2(t+s) = \operatorname{Var}(X(t+s)) = \operatorname{Var}(X(t+s)-X(t)) + \operatorname{Var}(X(t)) = \sigma^2(s) + \sigma^2(t).$$

### 特征函数与扩散系数
计算 $X(t)$ 的特征函数 $\varphi(t, \lambda) = E[e^{i\lambda X(t)}]$。通过增量独立性和泰勒展开，可推导出微分方程：
$$\frac{d}{dt} \varphi(t, \lambda) = -\varphi(t, \lambda) \frac{\lambda^2}{2} D,$$
其中 $D = \lim_{s\to 0} \sigma^2(s)/s$ 为**扩散系数**。解得：
$$\varphi(t, \lambda) = e^{-\frac{\lambda^2}{2} D t}.$$
这表明 $X(t) \sim N(0, D t)$。

### 与随机游走的关系
布朗运动可视为对称随机游走在时间步长趋于零、步长适当缩放时的连续极限。

## 标准布朗运动的性质与变换

### 简单性质
设 $\{ B(t), t \ge 0 \}$ 为标准布朗运动，则：
1. 具有平稳独立增量。
2. 均值函数 $m_B(t) = 0$，协方差函数与相关函数相同：
   $$C_B(s,t) = R_B(s,t) = \min(s, t).$$
   ==此性质是证明一个过程是标准布朗运动的关键。==

**定理**：若随机过程 $\{ X(t), t \ge 0 \}$ 满足：
- 样本轨道连续（以概率1）且 $X(0)=0$；
- $E[X(t)]=0$；
- 对 $s \le t$，$R_X(s,t) = s$；
- $X(t)$ 是正态过程，
则 $\{ X(t) \}$ 是标准布朗运动。

### 布朗运动的变换
设 $B(t)$ 是标准布朗运动，$a>0$，则以下随机过程也是标准布朗运动：
1. **对称性**：$W(t) = -B(t), \quad t \ge 0$；
2. **时齐性/平移性**：$W(t) = B(t+a) - B(a), \quad t \ge 0$；
3. **尺度变换**：$W(t) = B(at)/\sqrt{a}, \quad t \ge 0$；
4. **时间反演**：$W(0)=0, \quad W(t)=tB(1/t), \quad t>0$；
5. **有限时间反演**：对固定 $T>0$，$W(t)=B(T-t)-B(T), \quad t \in [0,T]$。

**验证(3)**：计算协方差，对 $s \le t$，
$$\operatorname{Cov}(W(s), W(t)) = \frac{1}{a} \operatorname{Cov}(B(as), B(at)) = \frac{1}{a} \min(as, at) = \min(s, t).$$

## 首中时与反正弦律

### 首中时
对常数 $a$，定义**首中时**（或**首达时**）：
$$T_a = \inf \{ t > 0: B(t) = a \}.$$

**分布推导**：设 $a>0$，利用全概率公式和布朗运动的对称性（反射原理），可得：
$$P(T_a \le t) = 2 P(B(t) \ge a) = 2 \left[ 1 - \Phi\left( \frac{a}{\sqrt{t}} \right) \right],$$
其中 $\Phi$ 为标准正态分布函数。由对称性，$T_a$ 与 $T_{-a}$ 同分布，故对任意 $a \ne 0$，
$$F_{T_a}(t) = P(T_a \le t) = P(T_{|a|} \le t) = 2 \left[ 1 - \Phi\left( \frac{|a|}{\sqrt{t}} \right) \right] = \frac{2}{\sqrt{2\pi}} \int_{|a|/\sqrt{t}}^{\infty} e^{-u^2/2} du.$$
求导得密度函数：
$$f_{T_a}(t) = \frac{|a|}{t\sqrt{2\pi t}} e^{-a^2/(2t)}, \quad t>0.$$

**重要结论**：
1. **质点最终到达任意点 $a$ 的概率为1**：$\lim_{t \to \infty} P(T_a \le t) = 1$。
2. **到达任意非零点 $a$ 的平均时间为无穷大**：$E[T_a] = +\infty$。

### 最大值过程
定义 $[0,t]$ 内的最大值过程：
$$M_t = \sup_{0 \le s \le t} B(s).$$
对 $a>0$，事件 $\{ M_t \ge a \}$ 等价于 $\{ T_a \le t \}$，故
$$P(M_t \ge a) = P(T_a \le t) = 2 \left[ 1 - \Phi\left( \frac{a}{\sqrt{t}} \right) \right].$$
其期望为：
$$E[M_t] = \int_0^\infty P(M_t > a) da = \int_0^\infty P(T_a < t) da = \sqrt{\frac{2t}{\pi}}.$$

### 反正弦律
用 $N(a, a+b]$ 表示质点在时间段 $(a, a+b]$ 内访问 $0$ 的次数。可以证明，对 $b > a > 0$，
$$P(N(a, b] = 0) = \frac{2}{\pi} \arcsin \sqrt{\frac{a}{b}}.$$
特别地，对 $t>0$ 和 $x \in (0,1)$，令 $a=tx, b=t(1-x)$，则
$$P(N(tx, t] = 0) = \frac{2}{\pi} \arcsin \sqrt{x}.$$
==反正弦律描述了布朗运动在特定时间段内不返回原点的概率，其分布服从反正弦分布，而非均匀分布。==

**定理总结**：
1. $P(T_a \le t) = P(T_{|a|} \le t) = 2P(B(t) \ge |a|)$；
2. 对 $a \ge 0$，$P(M_t \ge a) = P(T_a \le t)$；
3. 对 $a \ne 0$，$E[T_a] = \infty$；
4. 对 $b > a > 0$，$P(N(a, b] = 0) = \frac{2}{\pi} \arcsin \sqrt{a/b}$。

## 布朗桥与条件分布

### 布朗桥
设 $B(t)$ 为标准布朗运动，定义**布朗桥**：
$$X(t) = B(t) - t B(1), \quad t \in [0,1].$$
其均值为 $E[X(t)] = 0$，协方差函数（设 $0 \le s \le t \le 1$）为：
$$\operatorname{Cov}(X(s), X(t)) = s(1-t).$$
布朗桥是高斯过程，但不是标准布朗运动（其方差为 $t(1-t)$，而非 $t$）。

### 条件分布
已知 $B(t)=b$，求 $B(s)$ 的条件分布（$0<s<t$）。由于 $(B(s), B(t))$ 服从二元正态分布，利用条件分布公式可得：
$$E[B(s) \mid B(t)=b] = \frac{s}{t} b,$$
$$\operatorname{Var}(B(s) \mid B(t)=b) = s - \frac{s^2}{t} = \frac{s(t-s)}{t}.$$
因此，$B(s) \mid B(t)=b \sim N\left(\frac{s}{t}b, \frac{s(t-s)}{t}\right)$。
