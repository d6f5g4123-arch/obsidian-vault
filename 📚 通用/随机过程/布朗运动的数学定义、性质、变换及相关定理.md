# 布朗运动的数学定义、性质、变换及相关定理

## 1. 定义与基本性质

### 1.1 定义
若随机过程 $\{ X(t), t \ge 0 \}$ 满足以下四个条件，则称之为**布朗运动**（或**维纳过程**）：
1. $X(0) = 0$（通常约定从原点出发）。
2. 具有**独立增量**。
3. 对任意 $s, t \ge 0$，增量 $X(t) - X(s) \sim N(0, \sigma^2 |t - s|)$。
4. 样本轨道 $X(t)$ 是 $t$ 的连续函数（以概率1）。

当 $\sigma^2 = 1$ 时，称为**标准布朗运动**，通常记为 $\{ B(t), t \ge 0 \}$。

### 1.2 基本性质
布朗运动具有以下核心性质：
1. **独立增量性**：对任意 $0 \le t_1 < t_2 < \dots < t_n$，增量 $X(t_2)-X(t_1), \dots, X(t_n)-X(t_{n-1})$ 相互独立。
2. **平稳增量性**：增量 $X(t+s) - X(t)$ 的分布只依赖于时间差 $s$，与起始点 $t$ 无关。
3. **零均值**：$E[X(t)] = 0$。
4. **方差函数的连续性**：方差函数 $\sigma^2(t) = \operatorname{Var}(X(t))$ 是 $t$ 的连续函数（又称扩散函数）。

由独立增量性与平稳增量性可推导出方差函数的可加性：
$$\sigma^2(t+s) = \operatorname{Var}(X(t+s)) = \operatorname{Var}(X(t+s)-X(t)) + \operatorname{Var}(X(t)) = \sigma^2(s) + \sigma^2(t).$$

## 2. 特征函数与极限关系

### 2.1 特征函数推导
通过推导 $X(t)$ 的特征函数 $\varphi(t, \lambda) = E[e^{i\lambda X(t)}]$，可以证明 $X(t) \sim N(0, D t)$，其中 $D = \lim_{s\to 0} \sigma^2(s)/s$ 为扩散系数。最终得到：
$$\varphi(t, \lambda) = e^{-\frac{\lambda^2}{2} D t}.$$

### 2.2 与随机游走的关系
布朗运动可视为**对称随机游走**的连续时间极限。当随机游走的步长 $\Delta t \to 0$，且步幅 $\alpha$ 按 $\alpha \propto \sqrt{\Delta t}$ 缩放时，其极限过程即为布朗运动。

## 3. 标准布朗运动的性质
设 $\{ B(t), t \ge 0 \}$ 为标准布朗运动（$\sigma^2=1$），则：
1. 具有平稳独立增量。
2. 均值函数 $m_B(t) = 0$。
3. 协方差函数与相关函数相同：
   $$C_B(s,t) = R_B(s,t) = \min(s, t).$$
   证明（设 $s \le t$）：
   $$\begin{aligned} R_B(s,t) &= E[B(s)B(t)] = E[B(s)(B(t)-B(s)+B(s))] \\ &= E[B(s)(B(t)-B(s))] + E[B^2(s)] \\ &= E[B(s)]E[B(t)-B(s)] + \operatorname{Var}(B(s)) = s. \end{aligned}$$

**等价定义定理**：若随机过程 $\{ X(t), t \ge 0 \}$ 满足：
- 样本轨道连续（以概率1）且 $X(0)=0$；
- $E[X(t)]=0$；
- 对 $s \le t$，$R_X(s,t) = s$；
- $X(t)$ 是正态过程，
则 $\{ X(t) \}$ 是标准布朗运动。

**例题**：验证对于 $0 \le t_1 \le t_2 \le t_3 \le t_4$，增量正交：
$$E[(X(t_2)-X(t_1))(X(t_4)-X(t_3))] = E[X(t_2)X(t_4) - X(t_2)X(t_3) - X(t_1)X(t_4) + X(t_1)X(t_3)] = t_2 - t_2 - t_1 + t_1 = 0.$$
且 $E[(X(t)-X(s))^2] = t + s - 2s = t-s, \quad t \ge s$，故 $X(t)-X(s) \sim N(0, t-s)$。

## 4. 布朗运动的变换
设 $B(t)$ 是标准布朗运动，$a>0$，则以下随机过程也是标准布朗运动：
1. **对称变换**：$W(t) = -B(t), \quad t \ge 0$。
2. **时移变换**：$W(t) = B(t+a) - B(a), \quad t \ge 0$。
3. **尺度变换**：$W(t) = B(at)/\sqrt{a}, \quad t \ge 0$。
4. **时间反转变换**：$W(0)=0, \quad W(t)=tB(1/t), \quad t>0$。
5. **时间反演变换**（对固定 $T>0$）：$W(t)=B(T-t)-B(T), \quad t \in [0,T]$。

**验证尺度变换(3)**：计算协方差，对 $s \le t$，
$$\operatorname{Cov}(W(s), W(t)) = \frac{1}{a} \operatorname{Cov}(B(as), B(at)) = \frac{1}{a} \min(as, at) = \min(s, t).$$

## 5. 首中时与最大值过程

### 5.1 首中时定义与分布
对常数 $a$，定义**首中时**：
$$T_a = \inf \{ t > 0: B(t) = a \}.$$
其分布函数为（对任意 $a \ne 0$）：
$$F_{T_a}(t) = P(T_a \le t) = P(T_{|a|} \le t) = 2 \left[ 1 - \Phi\left( \frac{|a|}{\sqrt{t}} \right) \right] = \frac{2}{\sqrt{2\pi}} \int_{|a|/\sqrt{t}}^{\infty} e^{-u^2/2} du.$$
概率密度函数为：
$$f_{T_a}(t) = \frac{|a|}{t\sqrt{2\pi t}} e^{-a^2/(2t)}, \quad t>0.$$

**重要结论**：
1. 质点最终到达任意 $a \ne 0$ 的概率为1：$\lim_{t \to \infty} P(T_a \le t) = 1$。
2. ==但到达 $a$ 的平均时间是无穷的==：$E[T_a] = +\infty$。

### 5.2 最大值过程
定义 $[0,t]$ 内的最大值过程：
$$M_t = \sup_{0 \le s \le t} B(s).$$
对 $a>0$，事件 $\{ M_t \ge a \}$ 等价于 $\{ T_a \le t \}$，故
$$P(M_t \ge a) = P(T_a \le t) = 2 \left[ 1 - \Phi\left( \frac{a}{\sqrt{t}} \right) \right].$$
其期望为：
$$E[M_t] = \int_0^{\infty} P(M_t > a) da = \int_0^{\infty} P(T_a < t) da = \sqrt{\frac{2t}{\pi}}.$$

## 6. 反正弦律
用 $N(a, a+b]$ 表示质点在时间段 $(a, a+b]$ 内访问 $0$ 的次数。可以证明，对 $b > a > 0$，
$$P(N(a, b] = 0) = \frac{2}{\pi} \arcsin \sqrt{\frac{a}{b}}.$$
特别地，对 $t>0$ 和 $x \in (0,1)$，令 $a=tx, b=t(1-x)$，则
$$P(N(tx, t] = 0) = \frac{2}{\pi} \arcsin \sqrt{x}.$$
这反映了布朗运动在时间区间内不返回原点的概率分布，呈现出“反正弦”特征。

## 7. 布朗桥

### 7.1 定义与性质
设 $B(t)$ 为标准布朗运动，定义**布朗桥**：
$$X(t) = B(t) - t B(1), \quad t \in [0,1].$$
其性质如下：
- $E[X(t)] = 0$。
- 协方差函数（设 $0 \le s \le t \le 1$）：
  $$\begin{aligned} \operatorname{Cov}(X(s), X(t)) &= E[(B(s)-sB(1))(B(t)-tB(1))] \\ &= s - st - st + st = s(1-t). \end{aligned}$$
- 布朗桥是**高斯过程**，但不是标准布朗运动，因为其方差为 $t(1-t)$，而非 $t$。
- 布朗桥的端点固定：$X(0)=X(1)=0$。

### 7.2 条件分布
已知 $B(t)=b$，求 $B(s)$ 的条件分布（$0<s<t$）。由于 $(B(s), B(t))$ 服从二元正态分布，均值为0，协方差矩阵为 $\begin{pmatrix} s & s \\ s & t \end{pmatrix}$。利用条件分布公式：
$$\begin{aligned} E[B(s) \mid B(t)=b] &= \frac{s}{t} b, \\ \operatorname{Var}(B(s) \mid B(t)=b) &= s - \frac{s^2}{t} = \frac{s(t-s)}{t}. \end{aligned}$$
因此，$B(s) \mid \{B(t)=b\} \sim N\left(\frac{s}{t}b, \frac{s(t-s)}{t}\right)$。

## 8. 易混淆点与关联知识

### 8.1 易混淆点
1. **独立增量 vs 不相关增量**：对于布朗运动，独立增量意味着不相关，但反之不一定成立。布朗运动的增量是独立且正态的，因此不相关即独立。
2. **平稳增量**：指的是增量的分布只依赖于时间长度，不依赖于起始时刻。这并不意味着过程本身是平稳的（事实上，布朗运动的方差随时间线性增长，不是平稳过程）。
3. **首中时期望无穷**：虽然以概率1会击中任意水平 $a$，但平均等待时间是无穷大，这体现了布朗运动路径的极端不规则性。

### 8.2 关联知识
- [[随机过程]]：布朗运动是连续时间随机过程的核心例子。
- [[正态过程]]：布朗运动是一种特殊的高斯过程（正态过程）。
- [[鞅]]：标准布朗运动是一个**连续鞅**。
- [[伊藤积分]]：布朗运动是[[随机分析]]和[[伊藤积分]]理论的基础。
- [[扩散过程]]：布朗运动是最简单的扩散过程，其概率密度满足热传导方程（扩散方程）。
