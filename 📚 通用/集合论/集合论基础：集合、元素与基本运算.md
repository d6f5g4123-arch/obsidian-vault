# 集合论基础：集合、元素与基本运算

# 集合论基础

## 核心概念定义

**集合**是数学中最基本的概念之一，指具有某种特定性质的、确定的、互不相同的对象的全体。这些对象称为该集合的**元素**。

- 集合通常用大写字母 $A, B, C, \dots$ 表示。
- 元素通常用小写字母 $a, b, c, \dots$ 表示。
- 若对象 $a$ 是集合 $A$ 的元素，记作 $a \in A$（读作“$a$ 属于 $A$”）。
- 若对象 $a$ 不是集合 $A$ 的元素，记作 $a \notin A$。

## 详细说明

### 集合的表示法
1.  **列举法**：将集合的所有元素一一列举出来，用花括号括起。
    - 例如：$A = \{1, 2, 3, 4\}$，表示由数字1,2,3,4组成的集合。
2.  **描述法**：通过描述集合中元素的公共属性来定义集合。
    - 格式：$\{x | P(x)\}$，表示所有满足性质 $P$ 的对象 $x$ 构成的集合。
    - 例如：$B = \{x | x \text{ 是小于10的正偶数}\}$，等价于 $B = \{2, 4, 6, 8\}$。

### 特殊集合
- **空集**：不含任何元素的集合，记作 $\varnothing$ 或 $\{\}$。
- **全集**：在特定讨论范围内，包含所有可能元素的集合，通常记作 $U$。

### 集合间的基本关系
1.  **子集**：如果集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的**子集**，记作 $A \subseteq B$。
    - 若 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的**真子集**，记作 $A \subset B$。
    - 空集是任何集合的子集。
2.  **集合相等**：如果两个集合 $A$ 和 $B$ 包含完全相同的元素，则称它们相等，记作 $A = B$。这等价于 $A \subseteq B$ 且 $B \subseteq A$。

## 关键公式/运算

### 基本集合运算
1.  **并集**：由所有属于集合 $A$ **或** 属于集合 $B$ 的元素组成的集合。
    $$A \cup B = \{x | x \in A \text{ 或 } x \in B\}$$
2.  **交集**：由所有既属于集合 $A$ **又** 属于集合 $B$ 的元素组成的集合。
    $$A \cap B = \{x | x \in A \text{ 且 } x \in B\}$$
    - 如果 $A \cap B = \varnothing$，则称 $A$ 与 $B$ **不相交**。
3.  **差集（相对补集）**：由所有属于集合 $A$ 但**不属于**集合 $B$ 的元素组成的集合。
    $$A \setminus B = \{x | x \in A \text{ 且 } x \notin B\}$$
4.  **补集（绝对补集）**：在全集 $U$ 中，集合 $A$ 的补集由所有不属于 $A$ 的元素组成。
    $$A^c = \overline{A} = U \setminus A = \{x \in U | x \notin A\}$$

### 运算律
- **交换律**：$A \cup B = B \cup A$， $A \cap B = B \cap A$
- **结合律**：$(A \cup B) \cup C = A \cup (B \cup C)$， $(A \cap B) \cap C = A \cap (B \cap C)$
- **分配律**：
    - $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$
    - $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
- **德摩根定律**：
    - $\overline{A \cup B} = \overline{A} \cap \overline{B}$
    - $\overline{A \cap B} = \overline{A} \cup \overline{B}$

## 例题/应用

**例题1**：设全集 $U = \{1,2,3,4,5\}$， $A = \{1,3,5\}$， $B = \{2,3,5\}$。求 $A \cup B$， $A \cap B$， $A \setminus B$， $\overline{A}$。

**解**：
- $A \cup B = \{1,2,3,5\}$
- $A \cap B = \{3,5\}$
- $A \setminus B = \{1\}$
- $\overline{A} = U \setminus A = \{2,4\}$

**应用**：集合论是现代数学的通用语言和基础，是[[数理逻辑]]、[[关系与函数]]、[[概率论]]、[[数据库理论]]（如SQL查询中的并、交、差操作）等领域的基石。

## 易混淆点

1.  **元素与集合**：$a$ 和 $\{a\}$ 不同，前者是元素，后者是仅包含该元素的**单元素集合**。$a \in \{a\}$ 成立，但 $a = \{a\}$ 不成立。
2.  **子集与真子集**：==任何集合都是它自身的子集（$A \subseteq A$），但不是它自身的真子集。== 真子集要求至少有一个元素属于父集但不属于子集。
3.  **空集**：空集 $\varnothing$ 是唯一的，它是任何集合的子集。$\varnothing \in \{\varnothing\}$ 成立，但 $\varnothing \in \varnothing$ 不成立。
4.  **并集与交集**：并集是“或”的关系，取所有元素；交集是“且”的关系，取公共元素。当两个集合没有公共元素时，它们的交集是空集，而不是“0”或“无”。
