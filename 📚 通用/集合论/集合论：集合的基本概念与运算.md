# 集合论：集合的基本概念与运算

## 核心概念定义

**集合**是数学中最基本的概念之一，指具有某种特定性质的、确定的、互不相同的对象的全体。这些对象称为该集合的**元素**。

## 详细说明

### 集合的表示法
1.  **列举法**：将集合的所有元素一一列举出来，用花括号 `{}` 括起。
    - 例如：$A = \{1, 2, 3, 4\}$
2.  **描述法**：用集合中元素的共同特征来描述集合。
    - 例如：$B = \{x | x \text{是大于0的偶数}\}$

### 集合的关系
- **属于关系**：若对象 $a$ 是集合 $A$ 的元素，则称 $a$ 属于 $A$，记作 $a \in A$；否则记作 $a \notin A$。
- **子集**：如果集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的**子集**，记作 $A \subseteq B$。
    - 如果 $A \subseteq B$ 且 $B \subseteq A$，则称集合 $A$ 与 $B$ **相等**，记作 $A = B$。
    - 如果 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的**真子集**，记作 $A \subset B$。

### 集合的运算
1.  **并集**：由所有属于集合 $A$ 或属于集合 $B$ 的元素组成的集合，称为 $A$ 与 $B$ 的并集，记作 $A \cup B$。
    $$A \cup B = \{x | x \in A \text{ 或 } x \in B\}$$
2.  **交集**：由所有既属于集合 $A$ 又属于集合 $B$ 的元素组成的集合，称为 $A$ 与 $B$ 的交集，记作 $A \cap B$。
    $$A \cap B = \{x | x \in A \text{ 且 } x \in B\}$$
3.  **差集**（相对补集）：由所有属于集合 $A$ 但不属于集合 $B$ 的元素组成的集合，称为 $A$ 与 $B$ 的差集，记作 $A - B$ 或 $A \setminus B$。
    $$A - B = \{x | x \in A \text{ 且 } x \notin B\}$$
4.  **补集**（绝对补集）：设 $U$ 为全集，$A \subseteq U$，则由 $U$ 中所有不属于 $A$ 的元素组成的集合称为 $A$ 的补集，记作 $\complement_U A$ 或 $A^c$。
    $$A^c = \{x \in U | x \notin A\}$$

## 关键公式

以下是集合运算的基本恒等式（设 $A, B, C$ 为任意集合，$U$ 为全集）：

- **幂等律**：
  $$A \cup A = A, \quad A \cap A = A$$
- **交换律**：
  $$A \cup B = B \cup A, \quad A \cap B = B \cap A$$
- **结合律**：
  $$(A \cup B) \cup C = A \cup (B \cup C), \quad (A \cap B) \cap C = A \cap (B \cap C)$$
- **分配律**：
  $$A \cup (B \cap C) = (A \cup B) \cap (A \cup C), \quad A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$$
- **德摩根律**：
  $$(A \cup B)^c = A^c \cap B^c, \quad (A \cap B)^c = A^c \cup B^c$$
- **补集律**：
  $$A \cup A^c = U, \quad A \cap A^c = \varnothing, \quad (A^c)^c = A$$

## 例题/应用

**例题1：集合运算**
已知全集 $U = \{1, 2, 3, 4, 5\}$，$A = \{1, 3, 5\}$，$B = \{2, 3, 4\}$。
求：$A \cup B$， $A \cap B$， $A - B$， $B^c$。

**解**：
- $A \cup B = \{1, 2, 3, 4, 5\} = U$
- $A \cap B = \{3\}$
- $A - B = \{1, 5\}$
- $B^c = U - B = \{1, 5\}$

**应用**：集合论是现代数学的基石，是 [[数理逻辑]]、[[离散数学]]、[[概率论]]、[[计算机科学]]（尤其是数据库理论）等众多领域的基础。在编程中，集合数据结构（如 Python 的 `set`）直接应用了集合的运算。

```python
# Python 集合运算示例
A = {1, 3, 5}
B = {2, 3, 4}
print("并集:", A | B)  # {1, 2, 3, 4, 5}
print("交集:", A & B)  # {3}
print("差集(A-B):", A - B)  # {1, 5}
print("对称差集:", A ^ B)  # {1, 2, 4, 5}
```

## 易混淆点

1.  **元素与集合的关系 vs. 集合与集合的关系**：
    - $a \in \{a, b\}$ 是正确的（元素属于集合）。
    - $\{a\} \subseteq \{a, b\}$ 是正确的（子集关系）。
    - $\{a\} \in \{a, b\}$ 是错误的（一个集合本身不是另一个集合的元素，除非明确列出）。
2.  **空集**：不含任何元素的集合，记作 $\varnothing$ 或 $\{\}$。空集是任何集合的子集，即 $\varnothing \subseteq A$ 恒成立。
3.  **并集与交集优先级**：在无括号的情况下，通常认为交集运算优先级高于并集，但为避免混淆，==应始终使用括号明确运算顺序==。
