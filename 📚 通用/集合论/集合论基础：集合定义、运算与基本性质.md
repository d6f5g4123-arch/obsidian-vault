# 集合论基础：集合定义、运算与基本性质

# 集合论基础

## 核心概念定义

**集合**是数学中最基本的概念之一，通常指具有某种特定性质的、确定的、互不相同的对象的全体。这些对象称为该集合的**元素**。

- 若对象 $a$ 是集合 $A$ 的元素，记作 $a \in A$。
- 若对象 $b$ 不是集合 $A$ 的元素，记作 $b \notin A$。
- 集合通常用大写字母 $A, B, C, \dots$ 表示，元素用小写字母 $a, b, c, \dots$ 表示。
- 不含任何元素的集合称为**空集**，记作 $\varnothing$ 或 $\{\}$。

## 详细说明

### 集合的表示法
1.  **列举法**：将集合的所有元素一一列举出来，用花括号括起。
    - 例如：$A = \{1, 2, 3, 4\}$，表示由数字1,2,3,4组成的集合。
2.  **描述法**：通过描述集合中元素的公共属性来定义集合。
    - 格式：$\{x | P(x)\}$ 或 $\{x: P(x)\}$，表示“所有满足性质 $P$ 的对象 $x$ 构成的集合”。
    - 例如：$B = \{x | x \text{ 是小于10的正偶数}\}$，等价于 $B = \{2, 4, 6, 8\}$。

### 集合间的基本关系
- **子集**：如果集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的**子集**，记作 $A \subseteq B$。
    - 如果 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的**真子集**，记作 $A \subset B$（部分教材符号有差异）。
    - 空集是任何集合的子集：$\varnothing \subseteq A$。
- **相等**：如果两个集合 $A$ 和 $B$ 包含完全相同的元素，则称它们相等，记作 $A = B$。这等价于 $A \subseteq B$ 且 $B \subseteq A$。

### 集合的基本运算
1.  **并集**：由所有属于集合 $A$ **或** 属于集合 $B$ 的元素组成的集合，记作 $A \cup B$。
    $$A \cup B = \{x | x \in A \text{ 或 } x \in B\}$$
2.  **交集**：由所有既属于集合 $A$ **又** 属于集合 $B$ 的元素组成的集合，记作 $A \cap B$。
    $$A \cap B = \{x | x \in A \text{ 且 } x \in B\}$$
    如果 $A \cap B = \varnothing$，则称 $A$ 与 $B$ **不相交**。
3.  **差集**（相对补集）：由所有属于集合 $A$ 但**不属于**集合 $B$ 的元素组成的集合，记作 $A \setminus B$ 或 $A - B$。
    $$A \setminus B = \{x | x \in A \text{ 且 } x \notin B\}$$
4.  **补集**（绝对补集）：设 $U$ 为所讨论问题的**全集**（包含所有考虑元素的集合），则集合 $A$ 的补集是由所有属于 $U$ 但不属于 $A$ 的元素组成的集合，记作 $A^c$ 或 $\overline{A}$。
    $$A^c = U \setminus A = \{x \in U | x \notin A\}$$

## 关键公式/性质

集合运算满足一系列基本定律，这些定律与逻辑运算、布尔代数有深刻的联系。

### 基本恒等式
设 $A, B, C$ 为全集 $U$ 的子集。
- **幂等律**：$A \cup A = A$， $A \cap A = A$
- **交换律**：$A \cup B = B \cup A$， $A \cap B = B \cap A$
- **结合律**：$(A \cup B) \cup C = A \cup (B \cup C)$， $(A \cap B) \cap C = A \cap (B \cap C)$
- **分配律**：
    $$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$$
    $$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$$
- **同一律**：$A \cup \varnothing = A$， $A \cap U = A$
- **零律**：$A \cup U = U$， $A \cap \varnothing = \varnothing$
- **补集律**：$A \cup A^c = U$， $A \cap A^c = \varnothing$
- **双重否定律**：$(A^c)^c = A$
- **德摩根律**（非常重要）：
    $$(A \cup B)^c = A^c \cap B^c$$
    $$(A \cap B)^c = A^c \cup B^c$$
    ==德摩根律是考试和推理中的重点，它描述了并集补集等于补集交集，交集补集等于补集并集。==

## 例题/应用

**例题1：集合运算**
设全集 $U = \{1,2,3,4,5\}$， $A = \{1,3,5\}$， $B = \{2,3,4\}$。
求：$A \cup B$， $A \cap B$， $A \setminus B$， $B^c$。

**解**：
- $A \cup B = \{1,2,3,4,5\} = U$
- $A \cap B = \{3\}$
- $A \setminus B = \{1,5\}$
- $B^c = U \setminus B = \{1,5\}$

**例题2：证明德摩根律 $(A \cup B)^c = A^c \cap B^c$**
**证明**（元素法）：

设任意 $x \in (A \cup B)^c$。
根据补集定义，有 $x \notin (A \cup B)$。
根据并集定义，$x \notin (A \cup B)$ 意味着 $x \notin A$ **且** $x \notin B$。
再根据补集定义，$x \notin A$ 等价于 $x \in A^c$；$x \notin B$ 等价于 $x \in B^c$。
因此，$x \in A^c$ **且** $x \in B^c$，即 $x \in (A^c \cap B^c)$。

反之，设任意 $x \in (A^c \cap B^c)$。
根据交集定义，有 $x \in A^c$ **且** $x \in B^c$。
根据补集定义，$x \in A^c$ 等价于 $x \notin A$；$x \in B^c$ 等价于 $x \notin B$。
由于 $x$ 既不在 $A$ 中也不在 $B$ 中，根据并集定义，$x \notin (A \cup B)$。
再根据补集定义，$x \notin (A \cup B)$ 等价于 $x \in (A \cup B)^c$。

综上，$(A \cup B)^c$ 和 $A^c \cap B^c$ 包含完全相同的元素，因此 $(A \cup B)^c = A^c \cap B^c$。

## 易混淆点

1.  **元素与集合的关系 ($\in$) vs. 集合与集合的关系 ($\subseteq$)**：
    - $a \in \{a, b, c\}$ 是正确的（元素属于集合）。
    - $\{a\} \subseteq \{a, b, c\}$ 是正确的（集合是另一个集合的子集）。
    - $a \subseteq \{a, b, c\}$ 是错误的（元素不能是子集，除非 $a$ 本身被视作一个集合）。

2.  **空集 $\varnothing$ 的性质**：
    - 空集是任何集合的子集：$\varnothing \subseteq A$ 恒成立。
    - 空集没有元素：$\forall x, x \notin \varnothing$。
    - 空集是唯一的。

3.  **“或”在并集定义中的含义**：数学中的“或”是**可兼或**，即 $x \in A$ 或 $x \in B$ 包含了 $x$ 同时属于 $A$ 和 $B$ 的情况。

4.  **补集运算依赖于全集 $U$**：谈论补集 $A^c$ 时，必须明确当前讨论的**全集** $U$ 是什么，否则补集没有意义。

## 关联知识

- 集合论是现代数学的基石，为 [[数理逻辑]]、[[关系与函数]]、[[基数与序数]] 等高级概念提供基础。
- 集合运算与逻辑运算（与、或、非）一一对应，是 [[离散数学]] 和 [[布尔代数]] 的核心内容。
- 集合的 [[幂集]]（一个集合所有子集构成的集合）是理解集合大小和 [[基数]] 概念的关键。
- 集合论的思想广泛应用于 [[计算机科学]] 的数据结构（如集合类型）、数据库查询（SQL中的UNION, INTERSECT）和 [[算法]] 设计中。
