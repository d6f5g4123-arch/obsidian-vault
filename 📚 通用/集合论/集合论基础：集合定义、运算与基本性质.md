# 集合论基础：集合定义、运算与基本性质

# 集合论基础

## 1. 核心概念定义

**集合**是数学中最基本的概念之一，通常指具有某种特定性质的、确定的、互不相同的对象的全体。这些对象称为该集合的**元素**。

- 若对象 $a$ 是集合 $A$ 的元素，则记作 $a \in A$。
- 若对象 $b$ 不是集合 $A$ 的元素，则记作 $b \notin A$。
- 集合的表示方法：列举法（如 $A = \{1, 2, 3\}$）和描述法（如 $B = \{x | x > 0\}$）。

## 2. 详细说明

### 2.1 集合的基本关系
- **子集**：如果集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的**子集**，记作 $A \subseteq B$。
- **真子集**：如果 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的**真子集**，记作 $A \subset B$。
- **相等**：如果 $A \subseteq B$ 且 $B \subseteq A$，则集合 $A$ 与 $B$ 相等，记作 $A = B$。
- **空集**：不含任何元素的集合，记作 $\varnothing$ 或 $\{\}$。空集是任何集合的子集。

### 2.2 集合的基本运算
- **并集**：由所有属于集合 $A$ 或属于集合 $B$ 的元素组成的集合，记作 $A \cup B$。
  $$A \cup B = \{x | x \in A \text{ 或 } x \in B\}$$
- **交集**：由所有既属于集合 $A$ 又属于集合 $B$ 的元素组成的集合，记作 $A \cap B$。
  $$A \cap B = \{x | x \in A \text{ 且 } x \in B\}$$
- **差集**（补集）：由所有属于集合 $A$ 但不属于集合 $B$ 的元素组成的集合，记作 $A \setminus B$ 或 $A - B$。
  $$A \setminus B = \{x | x \in A \text{ 且 } x \notin B\}$$
- **补集**：在讨论一个**全集** $U$（包含所有讨论对象的集合）的前提下，集合 $A$ 的补集是由所有属于 $U$ 但不属于 $A$ 的元素组成的集合，记作 $A^c$ 或 $\overline{A}$。
  $$A^c = U \setminus A = \{x \in U | x \notin A\}$$

## 3. 关键公式/性质

以下是集合运算的基本定律（设 $A, B, C$ 为任意集合，$U$ 为全集）：

1. **幂等律**
   $$A \cup A = A, \quad A \cap A = A$$
2. **交换律**
   $$A \cup B = B \cup A, \quad A \cap B = B \cap A$$
3. **结合律**
   $$(A \cup B) \cup C = A \cup (B \cup C), \quad (A \cap B) \cap C = A \cap (B \cap C)$$
4. **分配律**
   $$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$$
   $$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$$
5. **德摩根定律**（De Morgan's Laws）==考试重点==
   $$(A \cup B)^c = A^c \cap B^c$$
   $$(A \cap B)^c = A^c \cup B^c$$
6. **与空集和全集的关系**
   $$A \cup \varnothing = A, \quad A \cap U = A$$
   $$A \cap \varnothing = \varnothing, \quad A \cup U = U$$

## 4. 例题/应用

**例题1**：设全集 $U = \{1, 2, 3, 4, 5\}$，$A = \{1, 2, 3\}$，$B = \{2, 4\}$。求 $A \cup B$， $A \cap B$， $A \setminus B$， $B^c$。

**解**：
- $A \cup B = \{1, 2, 3, 4\}$
- $A \cap B = \{2\}$
- $A \setminus B = \{1, 3\}$
- $B^c = U \setminus B = \{1, 3, 5\}$

**例题2**：用**德摩根定律**化简 $(A^c \cup B)^c$。

**解**：
$$(A^c \cup B)^c = (A^c)^c \cap B^c = A \cap B^c$$

## 5. 易混淆点

1. **元素与集合的关系** vs **集合与集合的关系**：$a \in A$ 表示元素与集合的“属于”关系；$A \subseteq B$ 表示集合与集合的“包含”关系。两者符号和意义完全不同。
2. **空集**：空集 $\varnothing$ 是一个没有任何元素的集合，但它本身是一个集合。$\{\varnothing\}$ 是一个含有一个元素（这个元素是空集）的集合，并非空集。
3. **补集**：补集是相对于一个特定的**全集** $U$ 而言的。没有指明全集时，补集没有意义。
4. **并集与交集**：并集是“或”的关系，取所有元素；交集是“且”的关系，取共同元素。

## 6. 关联知识

- [[子集]]：集合包含关系的核心概念。
- [[并集]]：集合的加法运算。
- [[交集]]：集合的乘法运算。
- [[幂集]]：一个集合所有子集构成的集合。
- [[笛卡尔积]]：集合间产生有序对的运算，是定义[[关系]]与[[函数]]的基础。
- [[映射]]：建立在集合基础上的函数概念。
- [[基数]]：集合中元素“个数”的推广，研究集合的大小。

集合论是现代数学的基石，为[[数理逻辑]]、[[实分析]]、[[拓扑学]]、[[概率论]]等众多分支提供了严格的语言和基础框架。
