# 集合论基础：集合定义、运算与基本性质

# 集合论基础

## 核心概念定义

**集合** 是数学中最基本的概念之一，指具有某种特定性质的、确定的、互不相同的对象的全体。这些对象称为该集合的 **[[元素]]**。

- 通常用大写字母 $A, B, C, \dots$ 表示集合。
- 用小写字母 $a, b, c, \dots$ 表示元素。
- 元素 $a$ 属于集合 $A$，记作 $a \in A$；不属于则记作 $a \notin A$。
- 集合的表示法：列举法（如 $A = \{1, 2, 3\}$）和描述法（如 $B = \{x | x \text{ 是偶数}\}$）。

## 详细说明与关键概念

### 1. 集合间的基本关系
- **[[子集]]**：若集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的子集，记作 $A \subseteq B$。若 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的 **真子集**，记作 $A \subset B$。
- **相等**：两个集合 $A$ 和 $B$ 相等，当且仅当 $A \subseteq B$ 且 $B \subseteq A$，记作 $A = B$。
- **空集**：不含任何元素的集合，记作 $\varnothing$ 或 $\{\}$。空集是任何集合的子集。

### 2. 集合的基本运算
- **[[并集]]**：集合 $A$ 和 $B$ 的并集是由所有属于 $A$ 或属于 $B$ 的元素组成的集合，记作 $A \cup B$。
  $$A \cup B = \{x | x \in A \text{ 或 } x \in B\}$$
- **交集**：集合 $A$ 和 $B$ 的交集是由所有既属于 $A$ 又属于 $B$ 的元素组成的集合，记作 $A \cap B$。
  $$A \cap B = \{x | x \in A \text{ 且 } x \in B\}$$
- **差集（补集）**：集合 $A$ 与 $B$ 的差集是由所有属于 $A$ 但不属于 $B$ 的元素组成的集合，记作 $A \setminus B$ 或 $A - B$。在给定全集 $U$ 下，$A$ 的补集记作 $A^c$ 或 $\overline{A}$。
  $$A \setminus B = \{x | x \in A \text{ 且 } x \notin B\}$$
  $$A^c = \{x \in U | x \notin A\}$$
- **对称差**：集合 $A$ 和 $B$ 的对称差是由所有属于 $A$ 或 $B$ 但不同时属于两者的元素组成的集合，记作 $A \bigtriangleup B$。
  $$A \bigtriangleup B = (A \setminus B) \cup (B \setminus A) = (A \cup B) \setminus (A \cap B)$$

## 关键公式与运算律

集合运算满足一系列基本定律，与逻辑运算、布尔代数有深刻联系。

- **幂等律**：$A \cup A = A$， $A \cap A = A$
- **交换律**：$A \cup B = B \cup A$， $A \cap B = B \cap A$
- **结合律**：$(A \cup B) \cup C = A \cup (B \cup C)$， $(A \cap B) \cap C = A \cap (B \cap C)$
- **分配律**：
  $$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$$
  $$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$$
- **德·摩根律**（对偶律）：
  $$(A \cup B)^c = A^c \cap B^c$$
  $$(A \cap B)^c = A^c \cup B^c$$
- **吸收律**：$A \cup (A \cap B) = A$， $A \cap (A \cup B) = A$

## 例题与应用

**例题1**：设全集 $U = \{1,2,3,4,5\}$， $A = \{1,3,5\}$， $B = \{2,3,4\}$。求 $A \cup B$， $A \cap B$， $A \setminus B$， $B^c$， $A \bigtriangleup B$。

**解**：
- $A \cup B = \{1,2,3,4,5\} = U$
- $A \cap B = \{3\}$
- $A \setminus B = \{1,5\}$
- $B^c = U \setminus B = \{1,5\}$
- $A \bigtriangleup B = (A \setminus B) \cup (B \setminus A) = \{1,5\} \cup \{2,4\} = \{1,2,4,5\}$

**应用**：集合论是数学的通用语言，是 **[[逻辑学]]**、**[[概率论]]**、**[[数据库理论]]**（SQL中的UNION, INTERSECT, EXCEPT）、**[[计算机科学]]**（数据结构、算法复杂度分析）等众多领域的基础。

## 易混淆点

1.  **元素与集合的关系（$\in$） vs. 集合与集合的关系（$\subseteq$）**：这是初学者最容易混淆的地方。
    - $a \in \{a, b, c\}$ 是正确的（元素属于集合）。
    - $\{a\} \subseteq \{a, b, c\}$ 是正确的（集合是另一个集合的子集）。
    - $a \subseteq \{a, b, c\}$ 是 **错误** 的（元素不能是子集）。

2.  **空集 $\varnothing$ 的特殊性**：
    - 空集是任何集合的子集：$\varnothing \subseteq A$ 恒成立。
    - 空集没有元素，但空集本身是一个集合对象。$\{\varnothing\}$ 是一个含有一个元素（该元素是空集）的集合，并非空集。

3.  **并集与交集对空集的运算**：
    - $A \cup \varnothing = A$
    - $A \cap \varnothing = \varnothing$

4.  **补集运算依赖于全集**：谈论 $A^c$ 时必须明确当前讨论的 **全集 $U$** 是什么，否则补集无意义。

## 关联知识

- **[[映射与函数]]**：函数可以定义为一种特殊的集合（有序对的集合）。
- **[[基数]]**：集合中元素“个数”的推广，用于比较无限集的大小。
- **[[公理化集合论]]**：为解决集合论悖论（如罗素悖论）而建立的严格公理体系（如ZFC系统）。
- **[[布尔代数]]**：集合代数（$\cup, \cap, ^c$）是布尔代数的典型实例。
- **[[谓词逻辑]]**：集合的描述法与谓词逻辑紧密相关，例如 $A = \{x | P(x)\}$。
