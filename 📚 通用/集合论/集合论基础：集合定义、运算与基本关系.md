# 集合论基础：集合定义、运算与基本关系

# 集合论基础

## 核心概念定义

**集合**是数学中最基本的概念之一，通常指具有某种特定性质的、确定的、互不相同的对象的全体。这些对象称为该集合的**元素**。

- 集合通常用大写字母 $A, B, C, \dots$ 表示。
- 元素通常用小写字母 $a, b, c, \dots$ 表示。
- 若 $a$ 是集合 $A$ 的元素，记作 $a \in A$；若不是，记作 $a \notin A$。

## 详细说明

### 集合的表示法
1.  **列举法**：将集合的所有元素一一列举出来，用花括号括起。
    - 例如：$A = \{1, 2, 3, 4\}$，表示由数字1,2,3,4组成的集合。
2.  **描述法**：通过描述集合中元素的公共属性来定义集合。
    - 格式：$\{x | P(x)\}$，其中 $P(x)$ 是关于 $x$ 的一个命题。
    - 例如：$B = \{x | x \text{ 是小于10的正偶数} \}$，即 $B = \{2, 4, 6, 8\}$。

### 特殊集合
- **空集**：不含任何元素的集合，记作 $\varnothing$ 或 $\{\}$。
- **全集**：在给定的问题或上下文中，所有可能元素的集合，通常记作 $U$。

## 关键关系与运算

### 基本关系
1.  **子集**：如果集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的**子集**，记作 $A \subseteq B$。
    - 如果 $A \subseteq B$ 且 $B \subseteq A$，则 $A = B$（集合相等）。
    - 如果 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的**真子集**，记作 $A \subset B$。
2.  **属于关系**：$\in$ 与 $\notin$，描述元素与集合的关系。

### 集合运算
1.  **并集**：由所有属于集合 $A$ **或** 属于集合 $B$ 的元素组成的集合，记作 $A \cup B$。
    $$A \cup B = \{x | x \in A \text{ 或 } x \in B\}$$
2.  **交集**：由所有既属于集合 $A$ **又** 属于集合 $B$ 的元素组成的集合，记作 $A \cap B$。
    $$A \cap B = \{x | x \in A \text{ 且 } x \in B\}$$
    - 如果 $A \cap B = \varnothing$，则称 $A$ 与 $B$ **不相交**。
3.  **差集**（相对补集）：由所有属于集合 $A$ 但**不属于**集合 $B$ 的元素组成的集合，记作 $A - B$ 或 $A \setminus B$。
    $$A - B = \{x | x \in A \text{ 且 } x \notin B\}$$
4.  **补集**（绝对补集）：设 $U$ 为全集，集合 $A$ 的补集由所有属于 $U$ 但不属于 $A$ 的元素组成，记作 $A^c$ 或 $\overline{A}$。
    $$A^c = U - A = \{x \in U | x \notin A\}$$

## 例题与应用

**例题1**：设 $A = \{1, 2, 3\}$，$B = \{2, 3, 4\}$，$U = \{1, 2, 3, 4, 5\}$。求：
1.  $A \cup B$
2.  $A \cap B$
3.  $A - B$
4.  $A^c$（相对于全集 $U$）

**解**：
1.  $A \cup B = \{1, 2, 3, 4\}$
2.  $A \cap B = \{2, 3\}$
3.  $A - B = \{1\}$
4.  $A^c = U - A = \{4, 5\}$

**应用**：集合论是现代数学的语言基础，是 [[数理逻辑]]、[[概率论]]（事件是样本空间的子集）、[[数据库理论]]（关系是元组的集合）、[[计算机科学]]（数据结构如集合、列表）等领域的基石。

## 易混淆点

1.  **$\in$ 与 $\subseteq$**：==这是初学者最易混淆的概念。==
    - $\in$ 表示“是...的元素”，连接**元素**与**集合**。
    - $\subseteq$ 表示“是...的子集”，连接**集合**与**集合**。
    - 例如：对于 $A = \{1, \{2\}\}$，有 $1 \in A$，但 $1 \subseteq A$ 不成立；$\{2\} \in A$，且 $\{\{2\}\} \subseteq A$。
2.  **空集的性质**：空集 $\varnothing$ 是任何集合的子集，即对任意集合 $A$，都有 $\varnothing \subseteq A$。
3.  **集合与元素**：集合本身也可以作为另一个集合的元素。
