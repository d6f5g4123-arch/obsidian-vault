# 集合论基础：集合定义、运算与基本关系

# 集合论基础

## 核心概念定义

**集合**是数学中最基本的概念之一，通常指具有某种特定性质的、确定的、互异的对象的全体。这些对象称为该集合的**元素**。

- 集合通常用大写字母 $A, B, C, \dots$ 表示。
- 元素通常用小写字母 $a, b, c, \dots$ 表示。
- 若对象 $a$ 是集合 $A$ 的元素，记作 $a \in A$（属于）。
- 若对象 $a$ 不是集合 $A$ 的元素，记作 $a \notin A$（不属于）。

集合的表示方法主要有两种：
1.  **列举法**：将元素一一列出，用花括号括起。例如，小于5的自然数集合：$\{1, 2, 3, 4\}$。
2.  **描述法**：用确定的条件表示集合中元素的公共属性。例如，小于5的自然数集合：$\{x | x \in \mathbb{N}, x < 5\}$。

## 详细说明与关键关系

### 集合间的基本关系
1.  **子集**：如果集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的**子集**，记作 $A \subseteq B$ 或 $B \supseteq A$。
    - 若 $A \subseteq B$ 且 $B \subseteq A$，则称 $A$ 与 $B$ **相等**，记作 $A = B$。
    - 若 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的**真子集**，记作 $A \subset B$（或 $A \subsetneq B$）。

2.  **空集**：不含任何元素的集合，记作 $\varnothing$ 或 $\{\}$。空集是任何集合的子集。

3.  **全集**：在特定讨论中，所涉及的所有元素构成的集合，通常记作 $U$ 或 $I$。

### 集合的基本运算
集合有三种基本运算，它们构成了集合代数的核心：
1.  **并集**：由所有属于集合 $A$ **或** 属于集合 $B$ 的元素组成的集合，记作 $A \cup B$。
    $$A \cup B = \{x | x \in A \text{ 或 } x \in B\}$$

2.  **交集**：由所有既属于集合 $A$ **又** 属于集合 $B$ 的元素组成的集合，记作 $A \cap B$。
    $$A \cap B = \{x | x \in A \text{ 且 } x \in B\}$$
    如果 $A \cap B = \varnothing$，则称 $A$ 与 $B$ **不相交**。

3.  **补集**（差集）：由所有属于集合 $A$ 但**不属于**集合 $B$ 的元素组成的集合，记作 $A \setminus B$ 或 $A - B$。
    $$A \setminus B = \{x | x \in A \text{ 且 } x \notin B\}$$
    在给定全集 $U$ 的前提下，集合 $A$ 的**绝对补集**记作 $A^c$ 或 $\overline{A}$，定义为 $U \setminus A$。

## 关键公式（集合恒等式）
以下恒等式对任意集合 $A, B, C$ 成立（假设运算在全集 $U$ 下进行）：

- **幂等律**：
  $A \cup A = A$
  $A \cap A = A$
- **交换律**：
  $A \cup B = B \cup A$
  $A \cap B = B \cap A$
- **结合律**：
  $(A \cup B) \cup C = A \cup (B \cup C)$
  $(A \cap B) \cap C = A \cap (B \cap C)$
- **分配律**：
  $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$ ==（重点）==
  $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$ ==（重点）==
- **德·摩根律**：
  $(A \cup B)^c = A^c \cap B^c$ ==（重点，易错点）==
  $(A \cap B)^c = A^c \cup B^c$ ==（重点，易错点）==
- **吸收律**：
  $A \cup (A \cap B) = A$
  $A \cap (A \cup B) = A$
- **与空集/全集的关系**：
  $A \cup \varnothing = A$
  $A \cap U = A$
  $A \cup U = U$
  $A \cap \varnothing = \varnothing$
  $A \cup A^c = U$
  $A \cap A^c = \varnothing$

## 例题/应用

**例题1**：设全集 $U = \{1,2,3,4,5\}$，$A = \{1,3,5\}$，$B = \{2,3,4\}$。求 $A \cup B$， $A \cap B$， $A \setminus B$， $B^c$。

**解**：
- $A \cup B = \{1,2,3,4,5\} = U$
- $A \cap B = \{3\}$
- $A \setminus B = \{1,5\}$
- $B^c = U \setminus B = \{1,5\}$

**例题2（证明）**：用定义证明德·摩根律之一：$(A \cup B)^c = A^c \cap B^c$。

**证明**：
设任意元素 $x$。
$x \in (A \cup B)^c \iff x \notin (A \cup B) \iff x \notin A \text{ 且 } x \notin B \iff x \in A^c \text{ 且 } x \in B^c \iff x \in (A^c \cap B^c)$。
由集合相等的定义，得证。

## 易混淆点
1.  **“属于”($\in$) 与 “包含于”($\subseteq$)**：前者描述元素与集合的关系，后者描述集合与集合的关系。例如，$1 \in \{1,2\}$ 正确，但 $1 \subseteq \{1,2\}$ 错误。$\{1\} \subseteq \{1,2\}$ 正确。
2.  **空集的性质**：空集 $\varnothing$ 没有元素，但 $\varnothing$ 本身是一个集合。$\{\varnothing\}$ 是含有一个元素（该元素是空集）的集合，并非空集。
3.  **德·摩根律的符号**：补集符号“c”作用于并集时，要变为交集，且每个集合都要取补，反之亦然。切忌只给其中一个集合取补。
4.  **交并运算的优先级**：在无括号时，通常认为交集运算优先级高于并集运算，类似于乘法优先于加法。为避免混淆，应多用括号。
