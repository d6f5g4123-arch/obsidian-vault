# 集合论基础：定义、运算与核心概念

# 集合论基础

## 核心概念定义

**集合**是数学中最基本的概念之一，指具有某种特定性质的、确定的、互不相同的对象的全体。这些对象称为集合的**元素**。

- 通常用大写字母 $A, B, C, \dots$ 表示集合。
- 用小写字母 $a, b, c, \dots$ 表示元素。
- 若 $a$ 是集合 $A$ 的元素，记作 $a \in A$；若不是，记作 $a \notin A$。
- 集合的表示法：列举法（如 $A = \{1, 2, 3\}$）和描述法（如 $B = \{x | x > 0\}$）。

## 详细说明

### 集合间的基本关系
1.  **[[子集]]**：若集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的子集，记作 $A \subseteq B$。
    - 若 $A \subseteq B$ 且 $B \subseteq A$，则 $A = B$。
    - 若 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的**真子集**，记作 $A \subset B$。
2.  **空集**：不含任何元素的集合，记作 $\varnothing$ 或 $\{\}$。空集是任何集合的子集。
3.  **全集**：在特定讨论范围内，包含所有元素的集合，通常记作 $U$。

### 集合的基本运算
1.  **[[并集]]**：由所有属于集合 $A$ **或** 属于集合 $B$ 的元素组成的集合，记作 $A \cup B$。
    $$A \cup B = \{x | x \in A \text{ 或 } x \in B\}$$
2.  **交集**：由所有既属于集合 $A$ **又** 属于集合 $B$ 的元素组成的集合，记作 $A \cap B$。
    $$A \cap B = \{x | x \in A \text{ 且 } x \in B\}$$
3.  **差集（相对补集）**：由所有属于集合 $A$ 但**不属于**集合 $B$ 的元素组成的集合，记作 $A - B$ 或 $A \setminus B$。
    $$A - B = \{x | x \in A \text{ 且 } x \notin B\}$$
4.  **补集（绝对补集）**：在全集 $U$ 中，所有不属于集合 $A$ 的元素组成的集合，记作 $A^c$ 或 $\overline{A}$。
    $$A^c = U - A = \{x \in U | x \notin A\}$$

## 关键公式/运算律

集合运算满足一系列运算律，与逻辑运算、布尔代数相通。

- **交换律**：$A \cup B = B \cup A$， $A \cap B = B \cap A$
- **结合律**：$(A \cup B) \cup C = A \cup (B \cup C)$， $(A \cap B) \cap C = A \cap (B \cap C)$
- **分配律**：$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$， $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
- **德摩根律**：$(A \cup B)^c = A^c \cap B^c$， $(A \cap B)^c = A^c \cup B^c$
- **幂等律**：$A \cup A = A$， $A \cap A = A$
- **吸收律**：$A \cup (A \cap B) = A$， $A \cap (A \cup B) = A$

## 例题/应用

**例题1**：设全集 $U = \{1,2,3,4,5\}$， $A = \{1,3,5\}$， $B = \{2,3,4\}$。求 $A \cup B$， $A \cap B$， $A - B$， $B^c$。

**解**：
- $A \cup B = \{1,2,3,4,5\} = U$
- $A \cap B = \{3\}$
- $A - B = \{1,5\}$
- $B^c = U - B = \{1,5\}$

**应用**：集合论是现代数学的通用语言和基础，广泛应用于：
1.  定义数学对象（如数、函数、空间）。
2.  计算机科学（数据库查询、编程语言类型系统）。
3.  逻辑学与概率论（事件空间）。

## 易混淆点

1.  **元素与集合的关系 ($\in$) vs. 集合与集合的关系 ($\subseteq$)**：
    - $a \in \{a, b, c\}$ 正确。
    - $\{a\} \subseteq \{a, b, c\}$ 正确。
    - $a \subseteq \{a, b, c\}$ **错误**，因为 $a$ 是元素，不是集合（除非特别定义）。
2.  **空集 $\varnothing$ 的性质**：
    - $\varnothing \subseteq A$ 对任何集合 $A$ 都成立。
    - $\varnothing \in \{\varnothing\}$ 成立，因为 $\{\varnothing\}$ 是一个以空集为唯一元素的集合。
    - $\varnothing \in \varnothing$ **不成立**，因为空集没有元素。
3.  **==集合的互异性==**：集合中的元素是互不相同的。$\{1,2,2,3\}$ 和 $\{1,2,3\}$ 表示同一个集合。
