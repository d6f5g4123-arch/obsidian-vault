# 集合论基础：定义、运算与核心概念

# 集合论基础

## 核心概念定义

**集合**是数学中最基本的概念之一，指具有某种特定性质的、确定的、互异的对象的全体。这些对象称为该集合的**元素**。

- 若 $a$ 是集合 $A$ 的元素，记作 $a \in A$。
- 若 $a$ 不是集合 $A$ 的元素，记作 $a \notin A$。
- 不含任何元素的集合称为**空集**，记作 $\varnothing$ 或 $\{\}$。

## 详细说明

### 集合的表示法
1.  **列举法**：将集合的元素一一列出，用花括号括起来。例如，$A = \{1, 2, 3\}$。
2.  **描述法**：用集合中元素的共同特征来描述。例如，$B = \{x | x \text{ 是大于0的偶数}\}$。

### 集合间的基本关系
- **子集**：如果集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的**子集**，记作 $A \subseteq B$。
- **真子集**：如果 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的**真子集**，记作 $A \subset B$。
- **集合相等**：如果 $A \subseteq B$ 且 $B \subseteq A$，则称集合 $A$ 与 $B$ 相等，记作 $A = B$。

## 关键公式/运算

### 集合的基本运算
1.  **并集**：由所有属于集合 $A$ 或属于集合 $B$ 的元素组成的集合，记作 $A \cup B$。
    $$A \cup B = \{x | x \in A \text{ 或 } x \in B\}$$
2.  **交集**：由所有既属于集合 $A$ 又属于集合 $B$ 的元素组成的集合，记作 $A \cap B$。
    $$A \cap B = \{x | x \in A \text{ 且 } x \in B\}$$
3.  **差集**（相对补集）：由所有属于集合 $A$ 但不属于集合 $B$ 的元素组成的集合，记作 $A \setminus B$ 或 $A - B$。
    $$A \setminus B = \{x | x \in A \text{ 且 } x \notin B\}$$
4.  **补集**（绝对补集）：设 $U$ 为全集，$A \subseteq U$，则由 $U$ 中所有不属于 $A$ 的元素组成的集合称为 $A$ 的补集，记作 $A^c$ 或 $\overline{A}$。
    $$A^c = \{x \in U | x \notin A\}$$

## 例题/应用

**例题1**：设全集 $U = \{1, 2, 3, 4, 5\}$，$A = \{1, 3, 5\}$，$B = \{2, 3, 4\}$。求 $A \cup B$， $A \cap B$， $A \setminus B$， $B^c$。

**解**：
- $A \cup B = \{1, 2, 3, 4, 5\} = U$
- $A \cap B = \{3\}$
- $A \setminus B = \{1, 5\}$
- $B^c = U \setminus B = \{1, 5\}$

**应用**：集合论是现代数学的语言基础，广泛应用于[[概率论]]、[[数理逻辑]]、[[计算机科学]]（如数据库查询、编程语言理论）等领域。

## 易混淆点

1.  **元素与集合的关系**（$\in$） vs **集合与集合的关系**（$\subseteq$）。例如，$1 \in \{1, 2\}$ 是正确的，但 $1 \subseteq \{1, 2\}$ 是错误的。
2.  **空集是任何集合的子集**，即 $\varnothing \subseteq A$ 恒成立。==这是常考重点==。
3.  **$\{\varnothing\}$ 不是空集**，它是一个包含空集作为其唯一元素的集合。
