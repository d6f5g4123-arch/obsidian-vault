# 集合论基础：定义、运算与基本性质

## 核心概念定义

**集合**是数学中最基本的概念之一，指具有某种特定性质的、确定的、互异的对象的全体。这些对象称为该集合的**元素**。

- 集合通常用大写字母 $A, B, C, \dots$ 表示。
- 元素通常用小写字母 $a, b, c, \dots$ 表示。
- 若 $a$ 是集合 $A$ 的元素，记作 $a \in A$；若 $a$ 不是集合 $A$ 的元素，记作 $a \notin A$。

## 详细说明

### 集合的表示方法
1.  **列举法**：将集合的所有元素一一列出，用花括号括起来。例如，小于5的自然数集合：$\{1, 2, 3, 4\}$。
2.  **描述法**：通过描述集合中元素的公共属性来表示。例如，所有偶数的集合：$\{x | x = 2k, k \in \mathbb{Z}\}$。

### 特殊集合
- **空集**：不含任何元素的集合，记作 $\varnothing$ 或 $\{\}$。
- **全集**：在特定讨论范围内所有元素构成的集合，通常记作 $U$。

## 关键公式/运算

### 基本运算
1.  **[[子集]]**：若集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的子集，记作 $A \subseteq B$。
    - 若 $A \subseteq B$ 且 $B \subseteq A$，则 $A = B$。
    - 空集是任何集合的子集：$\varnothing \subseteq A$。
2.  **[[并集]]**：由所有属于 $A$ 或属于 $B$ 的元素组成的集合，记作 $A \cup B$。
    $$A \cup B = \{x | x \in A \text{ 或 } x \in B\}$$
3.  **交集**：由所有既属于 $A$ 又属于 $B$ 的元素组成的集合，记作 $A \cap B$。
    $$A \cap B = \{x | x \in A \text{ 且 } x \in B\}$$
4.  **补集**：在全集中，所有不属于 $A$ 的元素组成的集合，记作 $A^c$ 或 $\overline{A}$。
    $$A^c = \{x \in U | x \notin A\}$$
5.  **差集**：由所有属于 $A$ 但不属于 $B$ 的元素组成的集合，记作 $A \setminus B$。
    $$A \setminus B = \{x | x \in A \text{ 且 } x \notin B\}$$

### 运算律
- 交换律：$A \cup B = B \cup A$， $A \cap B = B \cap A$。
- 结合律：$(A \cup B) \cup C = A \cup (B \cup C)$， $(A \cap B) \cap C = A \cap (B \cap C)$。
- 分配律：$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$， $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$。
- 德·摩根律：$(A \cup B)^c = A^c \cap B^c$， $(A \cap B)^c = A^c \cup B^c$。

## 例题/应用

**例题1**：设全集 $U = \{1, 2, 3, 4, 5\}$， $A = \{1, 3, 5\}$， $B = \{2, 3, 4\}$。求 $A \cup B$， $A \cap B$， $A \setminus B$， $B^c$。

**解**：
- $A \cup B = \{1, 2, 3, 4, 5\} = U$
- $A \cap B = \{3\}$
- $A \setminus B = \{1, 5\}$
- $B^c = U \setminus B = \{1, 5\}$

**应用**：集合论是现代数学的通用语言，是 [[数理逻辑]]、[[概率论]]、[[计算机科学]]（尤其是数据库理论）等领域的基石。

## 易混淆点

1.  **元素与集合的关系**：$a \in \{a\}$ 是正确的，但 $a \subseteq \{a\}$ 通常是错误的（除非 $a$ 本身也是一个集合）。元素与集合是“属于”关系，集合与集合是“包含”关系。
2.  **空集与零**：空集 $\varnothing$ 是一个没有元素的集合，而数字 $0$ 是一个元素。$\varnothing \neq \{0\}$，也 $\neq 0$。
3.  **子集与真子集**：若 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的**真子集**，记作 $A \subset B$（部分教材符号有差异）。==注意区分 $\subseteq$ 和 $\subset$ 在不同教材中的定义。==
