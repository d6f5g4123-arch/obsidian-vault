# 集合论基础：定义、运算与基本性质

# 集合论基础

## 核心概念定义

**集合**是数学中最基本的概念之一，指具有某种特定性质的、确定的、互不相同的对象的全体。这些对象称为该集合的**元素**。

- 通常用大写字母 $A, B, C, \ldots$ 表示集合。
- 用小写字母 $a, b, c, \ldots$ 表示元素。
- 若元素 $a$ 属于集合 $A$，记作 $a \in A$；若不属于，记作 $a \notin A$。
- 集合的表示法：列举法（如 $A = \{1, 2, 3\}$）和描述法（如 $B = \{x | x > 0\}$）。

## 详细说明与关键运算

### 1. [[子集]]
如果集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的**子集**，记作 $A \subseteq B$。
- 如果 $A \subseteq B$ 且 $B \subseteq A$，则称 $A$ 与 $B$ **相等**，记作 $A = B$。
- 如果 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的**真子集**，记作 $A \subset B$。
- 空集 $\emptyset$ 是任何集合的子集。

### 2. [[并集]]
由所有属于集合 $A$ **或** 属于集合 $B$ 的元素组成的集合，称为 $A$ 与 $B$ 的并集，记作 $A \cup B$。
$$ A \cup B = \{x | x \in A \text{ 或 } x \in B\} $$

### 3. [[交集]]
由所有既属于集合 $A$ **又** 属于集合 $B$ 的元素组成的集合，称为 $A$ 与 $B$ 的交集，记作 $A \cap B$。
$$ A \cap B = \{x | x \in A \text{ 且 } x \in B\} $$
如果 $A \cap B = \emptyset$，则称 $A$ 与 $B$ **不相交**。

### 4. [[补集]]（差集）
由所有属于集合 $A$ 但**不属于**集合 $B$ 的元素组成的集合，称为 $A$ 与 $B$ 的差集（或 $B$ 相对于 $A$ 的补集），记作 $A \setminus B$ 或 $A - B$。
$$ A \setminus B = \{x | x \in A \text{ 且 } x \notin B\} $$
在给定**全集** $U$ 的语境下，集合 $A$ 的补集记作 $A^c$ 或 $\overline{A}$，即 $A^c = U \setminus A$。

### 5. [[幂集]]
一个集合 $A$ 的所有子集构成的集合，称为 $A$ 的幂集，记作 $\mathcal{P}(A)$ 或 $2^A$。
如果 $A$ 有 $n$ 个元素，则其幂集有 $2^n$ 个元素。

## 关键公式与运算律

集合运算满足一系列基本定律，与逻辑运算、布尔代数有深刻联系。

### 基本恒等式
1.  **交换律**: $A \cup B = B \cup A$, $A \cap B = B \cap A$
2.  **结合律**: $(A \cup B) \cup C = A \cup (B \cup C)$, $(A \cap B) \cap C = A \cap (B \cap C)$
3.  **分配律**: 
    $$ A \cup (B \cap C) = (A \cup B) \cap (A \cup C) $$
    $$ A \cap (B \cup C) = (A \cap B) \cup (A \cap C) $$
4.  **德·摩根律**（对偶原理）: 
    $$ (A \cup B)^c = A^c \cap B^c $$
    $$ (A \cap B)^c = A^c \cup B^c $$
    （在差集形式下：$A \setminus (B \cup C) = (A \setminus B) \cap (A \setminus C)$）
5.  **幂等律**: $A \cup A = A$, $A \cap A = A$
6.  **同一律**: $A \cup \emptyset = A$, $A \cap U = A$
7.  **零律**: $A \cup U = U$, $A \cap \emptyset = \emptyset$
8.  **补集律**: $A \cup A^c = U$, $A \cap A^c = \emptyset$, $(A^c)^c = A$

## 例题与应用

**例题1**：设全集 $U = \{1,2,3,4,5\}$, $A = \{1,3,5\}$, $B = \{2,3,4\}$。求 $A \cup B$, $A \cap B$, $A \setminus B$, $B^c$。

**解**:
- $A \cup B = \{1,2,3,4,5\} = U$
- $A \cap B = \{3\}$
- $A \setminus B = \{1,5\}$
- $B^c = U \setminus B = \{1,5\}$

**例题2**：证明分配律 $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$。

**证明思路**（常用方法）：
1.  证明左边集合是右边集合的子集：任取 $x \in A \cup (B \cap C)$，分情况讨论 $x \in A$ 或 $x \in B \cap C$，推导出 $x \in (A \cup B) \cap (A \cup C)$。
2.  证明右边集合是左边集合的子集：任取 $x \in (A \cup B) \cap (A \cup C)$，推导出 $x \in A \cup (B \cap C)$。
3.  由集合相等的定义，得证。

## 易混淆点

1.  **元素与集合的关系 ($\in$) vs. 集合与集合的关系 ($\subseteq$)**：
    - $a \in \{a, b\}$ 是正确的。
    - $\{a\} \subseteq \{a, b\}$ 是正确的。
    - $a \subseteq \{a, b\}$ 是错误的（除非 $a$ 本身被定义为一个集合）。
    - $\{a\} \in \{a, b\}$ 是错误的，除非集合 $\{a, b\}$ 中包含元素 $\{a\}$。

2.  **空集 $\emptyset$ 的性质**：
    - 空集是任何集合的子集：$\emptyset \subseteq A$ 恒成立。
    - 空集没有元素：$\forall x, x \notin \emptyset$。
    - 空集的幂集不是空集：$\mathcal{P}(\emptyset) = \{\emptyset\}$，这是一个包含一个元素（空集本身）的集合。

3.  **==并集与交集的优先级==**：在无括号的情况下，通常认为**交集优先于并集**，即 $A \cup B \cap C$ 应理解为 $A \cup (B \cap C)$。为避免混淆，==强烈建议使用括号==明确运算顺序。

## 关联知识

- **逻辑基础**：集合运算与命题逻辑中的“或”(∨)、“与”(∧)、“非”(¬)运算一一对应，[[德·摩根律]]在两者中形式一致。
- **[[函数与映射]]**：函数定义基于集合的笛卡尔积。
- **[[基数]]**：比较集合“大小”的理论，有限集、可数集、不可数集。
- **[[公理化集合论]]**：如ZFC公理系统，为数学奠定更严谨的基础。
- **计算机科学**：集合是数据库理论（关系代数）、形式语言与自动机、布尔检索等领域的基石。
