# 集合论基础：定义、运算与基本性质

# 集合论基础

## 核心概念定义

**集合**是数学中最基本的概念之一，指具有某种特定性质的、确定的、互不相同的对象的全体。这些对象称为该集合的**元素**。

- 若元素 $a$ 属于集合 $A$，记作 $a \in A$。
- 若元素 $a$ 不属于集合 $A$，记作 $a \notin A$。
- 不含任何元素的集合称为**空集**，记作 $\varnothing$ 或 $\{\}$。

## 详细说明

### 集合的表示法
1.  **列举法**：将集合的所有元素一一列举出来，用花括号括起。
    - 例如：$A = \{1, 2, 3, 4\}$。
2.  **描述法**：用确定的条件表示集合中元素具有的共同性质。
    - 例如：$B = \{x | x \text{ 是大于0的偶数} \}$。

### 集合间的基本关系
- **子集**：如果集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的**子集**，记作 $A \subseteq B$。
    - 若 $A \subseteq B$ 且 $B \subseteq A$，则 $A = B$。
    - 若 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的**真子集**，记作 $A \subset B$。
- **幂集**：一个集合 $A$ 的所有子集构成的集合，称为 $A$ 的**幂集**，记作 $P(A)$ 或 $2^A$。
    - 若 $A$ 有 $n$ 个元素，则 $P(A)$ 有 $2^n$ 个元素。

### 集合的基本运算
1.  **并集**：由所有属于集合 $A$ **或**属于集合 $B$ 的元素组成的集合。
    $$A \cup B = \{x | x \in A \text{ 或 } x \in B\}$$
2.  **交集**：由所有既属于集合 $A$ **又**属于集合 $B$ 的元素组成的集合。
    $$A \cap B = \{x | x \in A \text{ 且 } x \in B\}$$
3.  **差集**（相对补集）：由所有属于集合 $A$ 但**不属于**集合 $B$ 的元素组成的集合。
    $$A - B = A \setminus B = \{x | x \in A \text{ 且 } x \notin B\}$$
4.  **补集**（绝对补集）：设 $U$ 为全集，$A \subseteq U$，则 $A$ 的补集为 $U$ 中所有不属于 $A$ 的元素组成的集合。
    $$A^c = \bar{A} = U - A = \{x \in U | x \notin A\}$$
5.  **对称差**：由所有属于 $A$ 或属于 $B$，但不同时属于两者的元素组成的集合。
    $$A \oplus B = (A - B) \cup (B - A) = (A \cup B) - (A \cap B)$$

## 关键公式与运算律

### 集合恒等式（类比逻辑运算或布尔代数）
- **幂等律**：$A \cup A = A$， $A \cap A = A$。
- **交换律**：$A \cup B = B \cup A$， $A \cap B = B \cap A$。
- **结合律**：$(A \cup B) \cup C = A \cup (B \cup C)$， $(A \cap B) \cap C = A \cap (B \cap C)$。
- **分配律**：
    $$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$$
    $$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$$
- **德·摩根律**（对偶原理）：
    $$(A \cup B)^c = A^c \cap B^c$$
    $$(A \cap B)^c = A^c \cup B^c$$
- **吸收律**：$A \cup (A \cap B) = A$， $A \cap (A \cup B) = A$。

## 例题与应用

**例题1**：设全集 $U = \{1,2,3,4,5\}$， $A = \{1,3,5\}$， $B = \{2,3,4\}$。求 $A \cup B$， $A \cap B$， $A - B$， $B^c$。

**解**：
- $A \cup B = \{1,2,3,4,5\} = U$
- $A \cap B = \{3\}$
- $A - B = \{1,5\}$
- $B^c = U - B = \{1,5\}$

**应用**：集合论是现代数学的通用语言和基础，广泛应用于：
- 数学基础：实数系、函数定义域/值域。
- 计算机科学：数据库查询（SQL中的 `UNION`, `INTERSECT`）、数据结构（集合类型）。
- 概率论：事件空间、事件的并交补运算。
- 逻辑学：谓词逻辑的论域。

## 易混淆点

1.  **元素与集合的关系（$\in$） vs. 集合与集合的关系（$\subseteq$）**：
    - $a \in \{a, b, c\}$ 是正确的。
    - $\{a\} \subseteq \{a, b, c\}$ 是正确的。
    - $a \subseteq \{a, b, c\}$ 是错误的，因为 $a$ 是元素，不是集合（除非特别定义）。
2.  **空集 $\varnothing$ 的性质**：
    - 空集是任何集合的**子集**：$\varnothing \subseteq A$ 恒成立。
    - 空集是任何非空集合的**真子集**。
    - 空集的**幂集**不是空集：$P(\varnothing) = \{\varnothing\}$，这是一个包含一个元素（空集本身）的集合。
3.  **补集运算依赖于全集的选取**：谈论补集 $A^c$ 时，必须明确当前讨论的**全集** $U$ 是什么。
