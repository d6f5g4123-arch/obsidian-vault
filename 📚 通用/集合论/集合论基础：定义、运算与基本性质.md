# 集合论基础：定义、运算与基本性质

# 集合论基础

## 1. 核心概念定义

**集合**是数学中最基本的概念之一，指具有某种特定性质的、确定的、互不相同的对象的全体。这些对象称为该集合的**元素**。

- 通常用大写字母 $A, B, C$ 表示集合，用小写字母 $a, b, c$ 表示元素。
- 若元素 $a$ 属于集合 $A$，记作 $a \in A$；若不属于，记作 $a \notin A$。
- 集合的表示法：列举法（如 $A = \{1, 2, 3\}$）和描述法（如 $B = \{x | x > 0\}$）。

## 2. 详细说明：基本关系与运算

### 2.1 集合间的关系
- **子集**：如果集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的**子集**，记作 $A \subseteq B$。
- **真子集**：若 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的**真子集**，记作 $A \subset B$。
- **相等**：若 $A \subseteq B$ 且 $B \subseteq A$，则集合 $A$ 与 $B$ 相等，记作 $A = B$。

### 2.2 集合的运算
- **并集**：由所有属于集合 $A$ 或属于集合 $B$ 的元素组成的集合，记作 $A \cup B$。
  $$A \cup B = \{x | x \in A \text{ 或 } x \in B\}$$
- **交集**：由所有既属于集合 $A$ 又属于集合 $B$ 的元素组成的集合，记作 $A \cap B$。
  $$A \cap B = \{x | x \in A \text{ 且 } x \in B\}$$
- **差集**（相对补集）：由所有属于集合 $A$ 但不属于集合 $B$ 的元素组成的集合，记作 $A \setminus B$ 或 $A - B$。
  $$A \setminus B = \{x | x \in A \text{ 且 } x \notin B\}$$
- **补集**（绝对补集）：设 $U$ 为全集，集合 $A$ 的补集由所有属于 $U$ 但不属于 $A$ 的元素组成，记作 $A^c$ 或 $\overline{A}$。
  $$A^c = \{x \in U | x \notin A\}$$

## 3. 关键公式与性质

### 3.1 运算律
- **交换律**：$A \cup B = B \cup A$， $A \cap B = B \cap A$。
- **结合律**：$(A \cup B) \cup C = A \cup (B \cup C)$， $(A \cap B) \cap C = A \cap (B \cap C)$。
- **分配律**：
  $$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$$
  $$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$$
- **德摩根定律**（非常重要）：
  $$\overline{A \cup B} = \overline{A} \cap \overline{B}$$
  $$\overline{A \cap B} = \overline{A} \cup \overline{B}$$

## 4. 例题与应用

**例题1**：设全集 $U = \{1, 2, 3, 4, 5\}$， $A = \{1, 3, 5\}$， $B = \{2, 3, 4\}$。求 $A \cup B$， $A \cap B$， $A \setminus B$， $B^c$。

**解**：
- $A \cup B = \{1, 2, 3, 4, 5\} = U$
- $A \cap B = \{3\}$
- $A \setminus B = \{1, 5\}$
- $B^c = U \setminus B = \{1, 5\}$

**应用**：集合论是现代数学的基石，是定义[[函数]]、[[关系]]、[[概率论]]样本空间、[[数据库]]查询（SQL中的UNION, INTERSECT）等概念的基础。

## 5. 易混淆点

1.  **元素与集合的关系**（$\in$） vs. **集合与集合的关系**（$\subseteq$）。例如， $\{a\} \subseteq \{a, b\}$ 正确，但 $a \subseteq \{a, b\}$ 错误，应为 $a \in \{a, b\}$。
2.  **空集 $\emptyset$**：不含任何元素的集合。它是任何集合的**子集**，即 $\emptyset \subseteq A$ 恒成立。==空集不是“无”，它是一个确定的集合对象==。
3.  **“或”在并集定义中**：数学中的“或”是包含性的，即 $x \in A$ 或 $x \in B$ 意味着 $x$ 可以只在 $A$ 中，或只在 $B$ 中，或同时在 $A$ 和 $B$ 中。
