# 集合论基础：定义、运算与基本性质

# 集合论基础

## 核心概念定义

**集合**是数学中最基本的概念之一，指具有某种特定性质的、确定的、互不相同的对象的全体。这些对象称为集合的**元素**。

- 通常用大写字母 $A, B, C, \dots$ 表示集合。
- 用小写字母 $a, b, c, \dots$ 表示元素。
- 若 $a$ 是集合 $A$ 的元素，记作 $a \in A$；若不是，记作 $a \notin A$。
- 集合的表示法：列举法（如 $A = \{1, 2, 3\}$）和描述法（如 $B = \{x | x \text{ 是偶数}\}$）。

## 详细说明与关键运算

### 1. 子集与相等
- **子集**：若集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的**子集**，记作 $A \subseteq B$。
- **真子集**：若 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的**真子集**，记作 $A \subset B$。
- **集合相等**：若 $A \subseteq B$ 且 $B \subseteq A$，则集合 $A$ 与 $B$ 相等，记作 $A = B$。

### 2. 并集、交集与补集
- **并集**：由所有属于集合 $A$ **或** 属于集合 $B$ 的元素组成的集合，称为 $A$ 与 $B$ 的**并集**，记作 $A \cup B$。
  $$A \cup B = \{x | x \in A \text{ 或 } x \in B\}$$
- **交集**：由所有既属于集合 $A$ **又** 属于集合 $B$ 的元素组成的集合，称为 $A$ 与 $B$ 的**交集**，记作 $A \cap B$。
  $$A \cap B = \{x | x \in A \text{ 且 } x \in B\}$$
- **补集**（差集）：由所有属于集合 $A$ 但**不属于**集合 $B$ 的元素组成的集合，称为 $A$ 与 $B$ 的**补集**（或差集），记作 $A \setminus B$ 或 $A - B$。
  $$A \setminus B = \{x | x \in A \text{ 且 } x \notin B\}$$

### 3. 空集与全集
- **空集**：不含任何元素的集合，记作 $\varnothing$ 或 $\{\}$。空集是任何集合的子集。
- **全集**：在特定讨论中，所考虑的所有元素的集合，通常记作 $U$。对于任意集合 $A$，有 $A \subseteq U$。

## 关键公式与运算律

集合运算满足一系列基本定律，与逻辑运算、布尔代数有深刻联系。

```
交换律：
A ∪ B = B ∪ A
A ∩ B = B ∩ A

结合律：
(A ∪ B) ∪ C = A ∪ (B ∪ C)
(A ∩ B) ∩ C = A ∩ (B ∩ C)

分配律：
A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)
A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)

德·摩根律：
(A ∪ B)^c = A^c ∩ B^c
(A ∩ B)^c = A^c ∪ B^c
（其中 A^c 表示 A 在全集 U 中的补集）
```

## 例题与应用

**例题1：** 设 $A = \{1, 2, 3\}$, $B = \{2, 3, 4\}$，求 $A \cup B$, $A \cap B$, $A \setminus B$。

**解：**
- $A \cup B = \{1, 2, 3, 4\}$
- $A \cap B = \{2, 3\}$
- $A \setminus B = \{1\}$

**应用：** 集合论是现代数学的基石，是定义[[函数]]、[[关系]]、[[基数]]等概念的基础，广泛应用于计算机科学（数据库查询、类型理论）、逻辑学、概率论（事件是样本空间的子集）等领域。

## 易混淆点

1.  **元素与集合的关系（$\in$） vs. 集合与集合的关系（$\subseteq$）**：
    - $a \in \{a, b\}$ 是正确的（元素属于集合）。
    - $\{a\} \subseteq \{a, b\}$ 是正确的（子集关系）。
    - $a \subseteq \{a, b\}$ 是错误的，除非 $a$ 本身被定义为一个集合。

2.  **空集 $\varnothing$ 的性质**：
    - 空集是唯一的。
    - 空集是任何集合的子集：$\varnothing \subseteq A$ 恒成立。
    - 空集与任何集合的交集是空集：$A \cap \varnothing = \varnothing$。
    - 空集与任何集合的并集是该集合本身：$A \cup \varnothing = A$。

3.  **==“或”在并集定义中是“可兼或”==**：$x \in A \cup B$ 意味着 $x$ 至少属于 $A$ 和 $B$ 中的一个，==允许同时属于两者==。
