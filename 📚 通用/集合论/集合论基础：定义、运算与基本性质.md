# 集合论基础

## 核心概念定义

**集合**是数学中最基本的概念之一，指具有某种特定性质的、确定的、互不相同的对象的全体。这些对象称为该集合的**元素**。

- 通常用大写字母 $A, B, C$ 表示集合。
- 用小写字母 $a, b, c$ 表示元素。
- 元素 $a$ 属于集合 $A$，记作 $a \in A$；不属于则记作 $a \notin A$。
- 集合的表示法：列举法（如 $A = \{1, 2, 3\}$）和描述法（如 $B = \{x | x > 0\}$）。

## 详细说明

### 1. 集合间的基本关系

- **子集**：如果集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的**子集**，记作 $A \subseteq B$。
- **真子集**：如果 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的**真子集**，记作 $A \subset B$。
- **相等**：如果 $A \subseteq B$ 且 $B \subseteq A$，则集合 $A$ 与 $B$ 相等，记作 $A = B$。
- **空集**：不含任何元素的集合，记作 $\varnothing$ 或 $\{\}$。空集是任何集合的子集。

### 2. 集合的基本运算

- **并集**：由所有属于集合 $A$ **或** 属于集合 $B$ 的元素组成的集合，记作 $A \cup B$。
  $$A \cup B = \{x | x \in A \text{ 或 } x \in B\}$$
- **交集**：由所有既属于集合 $A$ **又** 属于集合 $B$ 的元素组成的集合，记作 $A \cap B$。
  $$A \cap B = \{x | x \in A \text{ 且 } x \in B\}$$
- **差集**：由所有属于集合 $A$ 但**不属于**集合 $B$ 的元素组成的集合，记作 $A \setminus B$ 或 $A - B$。
  $$A \setminus B = \{x | x \in A \text{ 且 } x \notin B\}$$
- **补集**：设 $U$ 为全集，集合 $A$ 的补集由所有属于 $U$ 但不属于 $A$ 的元素组成，记作 $A^c$ 或 $\overline{A}$。
  $$A^c = \{x \in U | x \notin A\}$$

## 关键公式与性质

集合运算满足一系列基本定律，是集合论推理的基础：

- **交换律**：$A \cup B = B \cup A$， $A \cap B = B \cap A$
- **结合律**：$(A \cup B) \cup C = A \cup (B \cup C)$， $(A \cap B) \cap C = A \cap (B \cap C)$
- **分配律**：$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$， $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
- **德摩根律**：$(A \cup B)^c = A^c \cap B^c$， $(A \cap B)^c = A^c \cup B^c$
- **幂等律**：$A \cup A = A$， $A \cap A = A$
- **吸收律**：$A \cup (A \cap B) = A$， $A \cap (A \cup B) = A$

## 例题/应用

**例题1**：设全集 $U = \{1,2,3,4,5\}$， $A = \{1,3,5\}$， $B = \{2,3,4\}$。求 $A \cup B$， $A \cap B$， $A \setminus B$， $B^c$。

**解**：
- $A \cup B = \{1,2,3,4,5\} = U$
- $A \cap B = \{3\}$
- $A \setminus B = \{1,5\}$
- $B^c = U \setminus B = \{1,5\}$

**应用**：集合论是现代数学的通用语言，是 [[数理逻辑]]、[[概率论]]、[[数据库理论]]（如SQL查询中的并、交、差操作）、[[计算机科学]]（如数据结构、类型系统）等众多领域的基础。

## 易混淆点

1.  **元素与集合的关系** vs **集合与集合的关系**：$a \in \{a\}$ 是正确的（元素属于集合），但 $a \subseteq \{a\}$ 是错误的，除非 $a$ 本身是一个集合。应使用 $\{a\} \subseteq \{a\}$。
2.  **空集的性质**：空集 $\varnothing$ 是唯一的，并且 $\varnothing \subseteq A$ 对任何集合 $A$ 都成立。但 $\varnothing \in A$ 不一定成立，除非 $A$ 明确包含了空集作为元素。
3.  **并集与交集的优先级**：在没有括号的情况下，通常认为交集 $\cap$ 的优先级高于并集 $\cup$，类似于乘法对加法的优先级。为避免混淆，==应多用括号明确运算顺序==。
