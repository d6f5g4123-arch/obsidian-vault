# 集合论基础：核心概念与基本运算

# 集合论基础

## 核心概念定义

**集合**是数学中最基本的概念之一，指具有某种特定性质的、确定的、互异的对象的全体。这些对象称为该集合的**元素**。

## 详细说明

### 集合的表示法
1.  **列举法**：将集合中的所有元素一一列举出来，用花括号 `{}` 括起。
    - 例如：$A = \{1, 2, 3, 4\}$
2.  **描述法**：通过描述集合中元素的共同特征来定义集合。
    - 例如：$B = \{x | x \text{是大于0的偶数}\}$

### 集合间的基本关系
1.  **子集**：如果集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的**子集**，记作 $A \subseteq B$。
    - 如果 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的**真子集**，记作 $A \subset B$。
2.  **相等**：如果 $A \subseteq B$ 且 $B \subseteq A$，则集合 $A$ 与 $B$ 相等，记作 $A = B$。

### 集合的基本运算
1.  **并集**：由所有属于集合 $A$ 或属于集合 $B$ 的元素组成的集合，称为 $A$ 与 $B$ 的**并集**，记作 $A \cup B$。
    - 定义：$A \cup B = \{x | x \in A \text{ 或 } x \in B\}$
2.  **交集**：由所有既属于集合 $A$ 又属于集合 $B$ 的元素组成的集合，称为 $A$ 与 $B$ 的**交集**，记作 $A \cap B$。
    - 定义：$A \cap B = \{x | x \in A \text{ 且 } x \in B\}$
3.  **补集/差集**：由所有属于集合 $A$ 但不属于集合 $B$ 的元素组成的集合，称为 $B$ 关于 $A$ 的**补集**（或 $A$ 与 $B$ 的**差集**），记作 $A \setminus B$ 或 $A - B$。
    - 定义：$A \setminus B = \{x | x \in A \text{ 且 } x \notin B\}$
    - 在给定全集 $U$ 的语境下，集合 $A$ 的**补集**记作 $A^c$ 或 $\overline{A}$，定义为 $U \setminus A$。

## 关键公式/性质

以下是集合运算的一些基本定律（设 $A, B, C$ 为任意集合，$U$ 为全集）：

- **幂等律**：
  $A \cup A = A$
  $A \cap A = A$
- **交换律**：
  $A \cup B = B \cup A$
  $A \cap B = B \cap A$
- **结合律**：
  $(A \cup B) \cup C = A \cup (B \cup C)$
  $(A \cap B) \cap C = A \cap (B \cap C)$
- **分配律**：
  $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$
  $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
- **德摩根律**：
  $(A \cup B)^c = A^c \cap B^c$
  $(A \cap B)^c = A^c \cup B^c$

## 例题/应用

**例题1**：设全集 $U = \{1,2,3,4,5\}$，$A = \{1,3,5\}$，$B = \{2,3,4\}$。求 $A \cup B$， $A \cap B$， $A \setminus B$， $B^c$。

**解**：
- $A \cup B = \{1,2,3,4,5\} = U$
- $A \cap B = \{3\}$
- $A \setminus B = \{1,5\}$
- $B^c = U \setminus B = \{1,5\}$

**应用**：集合论是现代数学的通用语言和基础，广泛应用于**逻辑学**、**计算机科学**（数据库查询、形式语言与自动机）、**概率论**（事件空间）、**实分析**（点集拓扑）等几乎所有数学分支。

## 易混淆点

1.  **元素与集合的关系**（$\in$） vs **集合与集合的关系**（$\subseteq$）：
    - $a \in A$ 表示对象 $a$ 是集合 $A$ 的一个元素。
    - $B \subseteq A$ 表示集合 $B$ 的所有元素都是集合 $A$ 的元素。
    - 例如：$\{1\} \subseteq \{1,2\}$ 正确，但 $1 \subseteq \{1,2\}$ 是错误的写法。
2.  **空集 $\emptyset$**：不包含任何元素的集合。空集是任何集合的**子集**，即 $\emptyset \subseteq A$ 恒成立。
3.  **并集与交集**：并集是“或”的关系，取所有元素；交集是“且”的关系，取共同元素。==注意区分逻辑连接词==。
