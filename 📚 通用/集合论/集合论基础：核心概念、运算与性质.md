# 集合论基础：核心概念、运算与性质

## 核心概念定义

**集合**是数学中最基本的概念之一，指具有某种特定性质的、确定的、互不相同的对象的全体。这些对象称为该集合的**元素**。

- 若对象 $a$ 是集合 $A$ 的元素，记作 $a \in A$。
- 若对象 $a$ 不是集合 $A$ 的元素，记作 $a \notin A$。
- 不含任何元素的集合称为**空集**，记作 $\varnothing$ 或 $\{\}$。

## 详细说明

### 集合的表示法
1. **列举法**：将元素一一列出，如 $A = \{1, 2, 3\}$。
2. **描述法**：通过元素的共同特征来描述，如 $B = \{x \mid x \text{ 是偶数}\}$。

### 集合间的关系
- **子集**：若集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的**子集**，记作 $A \subseteq B$。
- **真子集**：若 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的**真子集**，记作 $A \subset B$。
- **相等**：若 $A \subseteq B$ 且 $B \subseteq A$，则集合 $A$ 与 $B$ 相等，记作 $A = B$。

### 集合的运算
1. **并集**：由所有属于集合 $A$ 或属于集合 $B$ 的元素组成的集合，记作 $A \cup B$。
   $$A \cup B = \{x \mid x \in A \text{ 或 } x \in B\}$$
2. **交集**：由所有既属于集合 $A$ 又属于集合 $B$ 的元素组成的集合，记作 $A \cap B$。
   $$A \cap B = \{x \mid x \in A \text{ 且 } x \in B\}$$
3. **差集**：由所有属于集合 $A$ 但不属于集合 $B$ 的元素组成的集合，记作 $A \setminus B$ 或 $A - B$。
   $$A \setminus B = \{x \mid x \in A \text{ 且 } x \notin B\}$$
4. **补集**：设 $U$ 为全集，集合 $A$ 的补集由所有属于 $U$ 但不属于 $A$ 的元素组成，记作 $A^c$ 或 $\overline{A}$。
   $$A^c = \{x \in U \mid x \notin A\}$$

## 关键公式/性质

- 幂等律：$A \cup A = A$， $A \cap A = A$
- 交换律：$A \cup B = B \cup A$， $A \cap B = B \cap A$
- 结合律：$(A \cup B) \cup C = A \cup (B \cup C)$， $(A \cap B) \cap C = A \cap (B \cap C)$
- 分配律：$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$， $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
- 德·摩根定律：$(A \cup B)^c = A^c \cap B^c$， $(A \cap B)^c = A^c \cup B^c$

## 例题/应用

**例题1**：设 $A = \{1, 2, 3\}$， $B = \{2, 3, 4\}$， 求 $A \cup B$， $A \cap B$， $A \setminus B$。

**解**：
- $A \cup B = \{1, 2, 3, 4\}$
- $A \cap B = \{2, 3\}$
- $A \setminus B = \{1\}$

**例题2**：证明 $A \setminus (B \cup C) = (A \setminus B) \cap (A \setminus C)$。

**证明**：
\begin{align*}
x \in A \setminus (B \cup C) &\iff x \in A \text{ 且 } x \notin (B \cup C) \\
&\iff x \in A \text{ 且 } (x \notin B \text{ 且 } x \notin C) \\
&\iff (x \in A \text{ 且 } x \notin B) \text{ 且 } (x \in A \text{ 且 } x \notin C) \\
&\iff x \in (A \setminus B) \text{ 且 } x \in (A \setminus C) \\
&\iff x \in (A \setminus B) \cap (A \setminus C)
\end{align*}

## 易混淆点

1. **元素与集合**：$a \in \{a\}$ 是正确的，但 $a = \{a\}$ 是错误的。元素和包含该元素的单元素集合是不同的概念。
2. **空集**：空集 $\varnothing$ 是任何集合的子集，即 $\varnothing \subseteq A$ 恒成立。==空集是唯一的==。
3. **子集与真子集**：任何集合 $A$ 都是其自身的子集 ($A \subseteq A$)，但不是其自身的真子集。
4. **并集与交集**：$A \cup B$ 包含 $A$ 和 $B$ 的所有元素，而 $A \cap B$ 只包含它们共有的元素。若 $A \cap B = \varnothing$，则称 $A$ 与 $B$ **不相交**。
