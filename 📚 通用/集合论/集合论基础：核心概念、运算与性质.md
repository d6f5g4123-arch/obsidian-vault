# 集合论基础

## 核心概念定义

**集合**是数学中最基本的概念之一，指具有某种特定性质的、确定的、互不相同的对象的全体。这些对象称为该集合的**元素**。

- 若对象 $a$ 是集合 $A$ 的元素，记作 $a \in A$。
- 若对象 $b$ 不是集合 $A$ 的元素，记作 $b \notin A$。
- 不含任何元素的集合称为**空集**，记作 $\varnothing$ 或 $\{\}$。

## 详细说明

### 集合的表示法
1.  **列举法**：将集合的所有元素一一列出，用花括号括起来。
    - 例如：$A = \{1, 2, 3, 4\}$，表示由数字1、2、3、4组成的集合。
2.  **描述法**：通过描述集合中元素的共同特征来定义集合。
    - 一般形式：$\{x | P(x)\}$，表示所有满足性质 $P$ 的对象 $x$ 构成的集合。
    - 例如：$B = \{x | x \text{ 是小于5的正整数} \}$，等价于 $B = \{1, 2, 3, 4\}$。

### 子集与相等
- **子集**：如果集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的**子集**，记作 $A \subseteq B$。
    - 若 $A \subseteq B$ 且 $B \subseteq A$，则称集合 $A$ 与 $B$ **相等**，记作 $A = B$。
    - 若 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的**真子集**，记作 $A \subset B$（或 $A \subsetneq B$）。
- 空集是任何集合的子集：$\varnothing \subseteq A$。

## 关键公式/运算

### 集合的基本运算
1.  **并集**：由所有属于集合 $A$ 或属于集合 $B$ 的元素组成的集合，记作 $A \cup B$。
    $$A \cup B = \{x | x \in A \text{ 或 } x \in B\}$$
2.  **交集**：由所有既属于集合 $A$ 又属于集合 $B$ 的元素组成的集合，记作 $A \cap B$。
    $$A \cap B = \{x | x \in A \text{ 且 } x \in B\}$$
3.  **差集**（相对补集）：由所有属于集合 $A$ 但不属于集合 $B$ 的元素组成的集合，记作 $A \setminus B$ 或 $A - B$。
    $$A \setminus B = \{x | x \in A \text{ 且 } x \notin B\}$$
4.  **补集**（绝对补集）：设 $U$ 为全集，集合 $A$ 的补集是由 $U$ 中所有不属于 $A$ 的元素组成的集合，记作 $A^c$ 或 $\overline{A}$。
    $$A^c = U \setminus A = \{x \in U | x \notin A\}$$

### 运算律
- **交换律**：$A \cup B = B \cup A$，$A \cap B = B \cap A$
- **结合律**：$(A \cup B) \cup C = A \cup (B \cup C)$，$(A \cap B) \cap C = A \cap (B \cap C)$
- **分配律**：$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$，$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
- **德·摩根律**：$(A \cup B)^c = A^c \cap B^c$，$(A \cap B)^c = A^c \cup B^c$

## 例题/应用

**例题1**：设全集 $U = \{1, 2, 3, 4, 5\}$，$A = \{1, 3, 5\}$，$B = \{2, 3, 4\}$。求 $A \cup B$，$A \cap B$，$A \setminus B$，$B^c$。

**解**：
- $A \cup B = \{1, 2, 3, 4, 5\} = U$
- $A \cap B = \{3\}$
- $A \setminus B = \{1, 5\}$
- $B^c = U \setminus B = \{1, 5\}$

**应用**：集合论是现代数学的基石，是 [[数理逻辑]]、[[关系与函数]]、[[概率论]]、[[数据库理论]]（如SQL查询中的并、交、差操作）等众多领域的基础语言和工具。

## 易混淆点

1.  **元素与集合的关系（$\in$） vs. 集合与集合的关系（$\subseteq$）**：
    - $1 \in \{1, 2, 3\}$ 是正确的（元素属于集合）。
    - $\{1\} \subseteq \{1, 2, 3\}$ 是正确的（集合是另一个集合的子集）。
    - $1 \subseteq \{1, 2, 3\}$ 是错误的（元素和集合之间不能用子集关系）。
2.  **空集 $\varnothing$ 的特殊性**：
    - 空集是任何集合的子集。
    - 空集本身没有元素，但 $\{\varnothing\}$ 是一个含有一个元素（该元素是空集）的集合，因此 $\varnothing \in \{\varnothing\}$ 成立，且 $\varnothing \subseteq \{\varnothing\}$ 也成立。
    - $\varnothing \neq \{\varnothing\}$。
3.  **并集与交集的优先级**：在没有括号的情况下，通常认为交集运算优先于并集运算，但为避免混淆，==强烈建议使用括号明确运算顺序==。
