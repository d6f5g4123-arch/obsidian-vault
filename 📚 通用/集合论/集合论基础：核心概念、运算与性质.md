# 集合论基础

## 核心概念定义

**集合**是数学中最基本的概念之一，通常指具有某种特定性质的、确定的、互异的对象的全体。这些对象称为集合的 **[[元素]]**。

- 集合通常用大写字母 $A, B, C, \dots$ 表示。
- 元素通常用小写字母 $a, b, c, \dots$ 表示。
- 若元素 $a$ 属于集合 $A$，记作 $a \in A$；反之，记作 $a \notin A$。
- 集合的表示法：列举法（如 $A = \{1, 2, 3\}$）和描述法（如 $B = \{x | x \text{ 是偶数}\}$）。

## 详细说明

### 集合间的基本关系
1.  **[[子集]]**：若集合 $A$ 的所有元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的子集，记作 $A \subseteq B$。
    - 若 $A \subseteq B$ 且 $B \subseteq A$，则称 $A$ 与 $B$ 相等，记作 $A = B$。
    - 若 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的真子集，记作 $A \subset B$。
2.  **空集**：不含任何元素的集合，记作 $\varnothing$ 或 $\{\}$。空集是任何集合的子集。
3.  **全集**：在特定讨论范围内，包含所有元素的集合，通常记作 $U$。

### 集合的基本运算
1.  **[[并集]]**：由所有属于集合 $A$ 或属于集合 $B$ 的元素组成的集合，记作 $A \cup B$。
    $$A \cup B = \{x | x \in A \text{ 或 } x \in B\}$$
2.  **交集**：由所有既属于集合 $A$ 又属于集合 $B$ 的元素组成的集合，记作 $A \cap B$。
    $$A \cap B = \{x | x \in A \text{ 且 } x \in B\}$$
3.  **差集（补集）**：由所有属于集合 $A$ 但不属于集合 $B$ 的元素组成的集合，记作 $A - B$ 或 $A \setminus B$。
    $$A - B = \{x | x \in A \text{ 且 } x \notin B\}$$
    特别地，在全集 $U$ 下，集合 $A$ 的补集记作 $A^c$ 或 $\overline{A}$，即 $A^c = U - A$。
4.  **对称差**：由所有属于 $A$ 或属于 $B$，但不同时属于两者的元素组成的集合，记作 $A \bigtriangleup B$。
    $$A \bigtriangleup B = (A - B) \cup (B - A) = (A \cup B) - (A \cap B)$$

## 关键公式/性质

集合运算满足一系列定律，与逻辑运算、布尔代数有深刻联系。

- **幂等律**：$A \cup A = A$， $A \cap A = A$
- **交换律**：$A \cup B = B \cup A$， $A \cap B = B \cap A$
- **结合律**：$(A \cup B) \cup C = A \cup (B \cup C)$， $(A \cap B) \cap C = A \cap (B \cap C)$
- **分配律**：$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$， $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
- **德·摩根律**：$(A \cup B)^c = A^c \cap B^c$， $(A \cap B)^c = A^c \cup B^c$
- **吸收律**：$A \cup (A \cap B) = A$， $A \cap (A \cup B) = A$

## 例题/应用

**例题1**：设全集 $U = \{1, 2, 3, 4, 5\}$， $A = \{1, 3, 5\}$， $B = \{2, 3, 4\}$。求 $A \cup B$， $A \cap B$， $A - B$， $B^c$。

**解**：
- $A \cup B = \{1, 2, 3, 4, 5\} = U$
- $A \cap B = \{3\}$
- $A - B = \{1, 5\}$
- $B^c = U - B = \{1, 5\}$

**应用**：集合论是数学的基石，为[[数理逻辑]]、[[关系与函数]]、[[概率论]]（样本空间、事件）、[[数据库理论]]（关系模型）、[[计算机科学]]（数据结构、形式语言与自动机）等提供了严格的描述语言和基础框架。

## 易混淆点

1.  **元素与集合的关系（$\in$） vs. 集合与集合的关系（$\subseteq$）**：
    - $a \in \{a, b\}$ 是正确的（元素属于集合）。
    - $\{a\} \subseteq \{a, b\}$ 是正确的（集合是子集）。
    - $a \subseteq \{a, b\}$ 是错误的（元素不能作为子集关系的一方）。
2.  **空集 $\varnothing$**：
    - $\varnothing$ 是任何集合的子集。
    - $\varnothing$ 不是 $\{\varnothing\}$ 的元素，但 $\varnothing$ 是 $\{\varnothing\}$ 的子集。即 $\varnothing \subseteq \{\varnothing\}$ 为真，但 $\varnothing \in \{\varnothing\}$ 也为真（注意后者集合里有一个元素，这个元素是空集本身）。
    - $\varnothing \in \varnothing$ 为假。
3.  **并集与交集**：并集是“或”的关系，取所有元素；交集是“且”的关系，取公共元素。==考试中常要求准确区分并计算。==
